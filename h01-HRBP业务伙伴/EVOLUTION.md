---

## 🔄 OODA进化循环

```
┌─────────────────────────────────────────────────────┐
│                  OODA 循环 (每日)                     │
│                                                       │
│  Observe（观察）  →  Orient（定向）  →                │
│  人才动态扫描        根因诊断+趋势判断                  │
│       ↑                                    ↓          │
│       │                                    │          │
│  Act（行动）      ←  Decide（决策）     ←             │
│  预警+方案+落地      优先级排序+方案选项                │
│       │                                                │
│       └────────→  Evolve（进化）←────────────────      │
│                   经验沉淀+方法论升级+预判能力提升       │
└─────────────────────────────────────────────────────┘
```

### 各阶段关键动作

| 阶段 | 动作 | 输出 |
|------|------|------|
| **Observe** | 人才动态扫描、组织健康监测、氛围感知 | 人才预警信号 + 组织健康快照 |
| **Orient** | 离职归因分析、团队诊断、趋势判断 | 诊断报告 + 风险评级 |
| **Decide** | 优先级排序、方案选项设计 | 方案A/B + 预期效果评估 |
| **Act** | 预警推送、招聘执行、方案落地 | 行动记录 + 执行反馈 |
| **Evolve** | 教训归档、方法论提炼、判断力校准 | 技能基因更新 + 预判模型升级 |

## 🧬 技能基因库

> 每一条基因 = 一次有价值的经验固化。经过验证的判断模式，沉淀为可复用的思维模型。

### 基因 #1：离职信号识别基因

```yaml
类型: 预警判断
创建场景: 源于华为HRBP经验——核心人才离职前3-6个月就会出现信号
核心模式: |
  核心人才离职六大信号：
  1. 工作投入度明显下降（不加班/不接额外任务）
  2. 频繁请假或迟到早退
  3. 对长期项目的热情降低
  4. 与上级/同侪的沟通频率减少
  5. 对薪酬/晋升话题异常敏感
  6. LinkedIn/招聘网站活跃度增加
  
  触发≥3个信号 → 立即启动保留面谈
  ≤2个信号 → 标记观察 + 月度复查
```

### 基因 #2：面试红线基因

```yaml
类型: 判断标准
创建场景: 源于阿里政委经验——一次错误的招聘让团队花了6个月消化
核心模式: |
  面试中的6条不可逾越红线（任一触发→不录用）：
  1. 简历造假（经历/学历/薪酬任一不实）
  2. 面试中贬低所有前雇主（不是一两家，是所有）
  3. 无法说出任何一次失败经历
  4. 对团队协作表现出明显的抵触
  5. 薪酬期望严重背离市场（>50%溢价且无特殊理由）
  6. 无法接受任何形式的反馈或质疑
  
  此外，面试官直觉的"不舒服感"即使无法量化，也必须记录为减分项。
置信度: 0.95
```

### 基因 #3：组织设计四原则基因

```yaml
类型: 设计原则
创建场景: 基于华为组织调整和字节OD的实战经验
核心模式: |
  任何组织架构调整必须满足四个原则：
  1. 战略适配：架构是否让业务跑得更快？（而非让权力更集中）
  2. 决策效率：决策链条是否≤3层？审批节点是否必要？
  3. 人才匹配：现有人才能否支撑新架构？缺口在哪里？
  4. 人性过渡：受影响的个体是否有体面的过渡方案？
  
  四原则任一不满足 → 方案退回重设计
置信度: 0.88
使用次数: 0（待实战验证）
创建日期: 2026-06-18
```

### 基因 #4：文化落地三步法基因

```yaml
类型: 方法框架
创建场景: 源于阿里政委海外本土化经验
核心模式: |
  文化落地不是贴标语，是三步法：
  1. 翻译：把抽象价值观翻译为具体行为标准
     例："客户第一"→"客户投诉24小时内响应并给出解决方案"
  2. 机制化：把行为标准嵌入考核/激励/晋升体系
     例：价值观考核占绩效权重20-30%
  3. 故事化：每个季度至少收集1个价值观故事
     例：谁在什么情况下做了什么事体现了什么价值观
     
  文化落地程度 = 翻译准确度 × 机制嵌入度 × 故事传播度
置信度: 0.85
使用次数: 0（待实战验证）
创建日期: 2026-06-18
```
