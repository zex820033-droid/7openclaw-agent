# 心跳清单 — HRBP/业务伙伴 h01 · 🤝

> **用途**：周期性自动巡检 | 人才预警 | 组织健康监测
> **原则**：人是最不可控的变量——宁可多问一句，不可漏掉一个信号。

---

## 🕐 每6小时（06:00 / 12:00 / 18:00 / 24:00）

### 1. 核心人才状态扫描
- [ ] 检查核心人才（T序列+L3+）是否有离职信号
  - 近期是否频繁请假/迟到
  - 工作投入度是否有明显下降
  - 是否在招聘网站更新简历
- [ ] 标记异常 → 推送预警至CEO

### 2. 关键岗位空缺状态
- [ ] 检查所有开放中的关键岗位招聘进度
- [ ] 空缺超过15天 → 推送提醒至招聘负责人
- [ ] 空缺超过30天 → 升级为P0，推送CEO

### 3. 团队氛围快检
- [ ] 各板块是否有异常的人事变动
- [ ] 是否有跨团队冲突升级的信号
- [ ] 记录氛围变化至 `data/team-climate-log.md`

---

## 🌅 每日（09:00）

### 招聘进度检查
- [ ] 核验昨日的面试安排与面试结果
- [ ] 检查面试评估表是否在24h内提交
- [ ] 确认候选人流程是否卡在某个环节
- [ ] 更新招聘看板

### 入职与离职
- [ ] 今日入职人员 → 确认融入计划已启动
- [ ] 今日离职/交接中人员 → 确认交接顺利
- [ ] 记录至 `data/onboarding-offboarding-log.md`

---

## 📅 每周（周五 17:00）

### 人才周报
- [ ] 本周招聘汇总：面试数/Offer数/入职数
- [ ] 本周离职汇总：离职人数/离职原因分类/关键人才预警
- [ ] 在途招聘周期分析：平均招聘周期是否超出目标
- [ ] 推送CEO《人才周报》

### 团队健康快报
- [ ] 各板块团队状态速览
- [ ] 本周重要人事动态（入职/离职/调动）
- [ ] 下周重点关注事项

---

## 📊 每月（月末最后一个工作日）

### 人才月报
- [ ] 月度招聘漏斗分析
- [ ] 月度离职率与离职归因
- [ ] 核心人才留存率计算
- [ ] 关键岗位空缺率统计

### 组织健康度月检
- [ ] 管理层级/管理幅度是否有异常
- [ ] 团队氛围整体评价
- [ ] 文化价值观落地情况抽查

---

## 📅 每半年度（3月 + 9月）

### 人才盘点启动
- [ ] 触发 `talent-nine-box` 技能
- [ ] 通知各业务板块准备绩效数据
- [ ] 安排校准会议日期
- [ ] 完成全量人才盘点 + 发展计划

---

## 🚨 熔断条件（触发立即上报，不等周期）

| 条件 | 阈值 | 动作 |
|------|------|------|
| 核心人才离职 | T序列/L3+任一提出离职 | 立即上报CEO + 1h内启动离职面谈 |
| 多人连续离职 | 同一团队连续3人/月离职 | 启动团队诊断 + 推送CEO |
| 关键岗位空缺 | 空缺 > 30天 | 升级为P0 + 推CEO讨论替代方案 |
| 劳资纠纷 | 任何劳动仲裁/法律风险 | 立即上报CEO + 启动法律合规流程 |
| 团队氛围危机 | 跨团队冲突升级到影响业务 | 启动紧急协调 + 上报CEO |
