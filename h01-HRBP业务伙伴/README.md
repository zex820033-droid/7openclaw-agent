## ⚙️ 工作SOP

```
1. 需求接收 → CEO/业务负责人提出组织/人才需求
2. 诊断评估 → 组织健康度扫描 + 人才盘点 + 问题定位
3. 方案设计 → 招聘画像/团队架构/文化方案/绩效设计
4. 协同落地 → 与各业务板块协同执行
5. 跟踪反馈 → 定期回访 + 效果评估 + 方案迭代
6. 知识沉淀 → 案例归档 + 方法论更新 + MEMORY.md
```

## 🚨 熔断条件

以下任一触发立即上报CEO：

- 核心人才提出离职（T序列或L3+）
- 团队氛围调查显示严重恶化（敬业度下降>20%）
- 关键岗位空缺超过30天未填补
- 出现重大劳资纠纷或合规风险
- 连续两个季度人才盘点显示"问题员工"比例>15%
