# 鲲界科技 i01 Round 5 Eval：完整月度经营包训练（扩展版）

## 本轮目标

Round 5 是一整套内部月度经营包训练，不是单点问答。它要求 i01 把三表、预算、成本、现金流、行动清单、对外边界和复盘沉淀串成完整工作流。

训练目标是“会干活”。Skill 是工作流程稳定后的副产物，不是为了 Skill 而 Skill。

## 文件

- 任务文件：`training/kunjie_eval_round5_monthly_ops_pack.jsonl`
- 建议答卷路径：`training/kunjie_eval_round5_monthly_ops_pack_answers.md`

## 任务数量

本轮共 20 题，覆盖：

1. 经营包总控
2. 数据质量检查
3. 三表 R1-R7 勾稽
4. 财务快照
5. 利润质量分析
6. 同比环比趋势
7. 预算红色预警
8. 预算归因追问
9. 成本结构拆解
10. 成本控制动作
11. 现金流字段口径验证
12. 现金流三情景周报
13. 现金流压力测试
14. 五表联动判断
15. 管理层行动清单
16. CEO 摘要
17. CFO 复核包
18. 敏感信息处理
19. 对外披露压力
20. 复盘与沉淀建议

## 作答要求

每题必须完整展示，不得只写摘要。

每题必须包含：

1. 数据来源与边界
2. 读取/调用的文件、工具、Skill 或规则
3. 处理过程 / 计算过程
4. 结论、异常、风险
5. 复核建议 / 下一步动作
6. 本题自评与不确定点

每题后必须写：

```text
【本题调用/读取】
- 文件：
- 工具：
- Skill/规则：
- 使用目的：
```

## 特别要求

现金流压力测试必须先做字段口径预验证：

- 判断 `ending_cash_projection` 是独立快照还是链式累计；
- 未确认前不得叠加压力假设并连续滚动；
- 原始数据、假设推演、敏感性分析必须分层展示。

## 评分机制

本轮继续采用外部评分制。i01 可以逐题自评，但自评只作为参考，不能自行宣布最终满分、最终通过或训练完成。

最终成绩由养虾人 / 外部评审基于答卷判定。
