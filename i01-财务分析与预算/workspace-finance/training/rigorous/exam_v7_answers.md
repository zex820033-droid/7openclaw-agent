### ② 假性超支拆解

**表面超支**: 1,155,000-980,000=+175,000(+17.9%) — ❌假性!

**产量效应**(合理): 1,140,000-980,000=**+160,000**(因产量从10k→12k件,合理)

### ③ 非经常性项目

- 设备紧急维修¥15,000 → 计入固定制造费，一次性事件。剔除后固定制造偏差=192,000-15,000-180,000=**-3,000**(实际是节约!)

**材料超支¥15,000归因**: 加急运费10,000+价格/用量差异5,000。采购管理问题是主因。

**人工"节约"¥12,000的背后**:产量+20%,人工总金额反而减少12,000.但5%调薪使单位人工成本上升。效率提升抵消了调薪效应——需要具体工时分析。

**变动制造费率隐含问题**:表面偏差为0,但实际费率¥16.70>预算¥16,效率改善(11,500h<12,000h)刚好补偿。隐含问题:费率上涨趋势。

**建议**:①停止紧急调货,建立备选供应商池 ②深入分析人工效率数据(正式工vs临时工)③设备维修费做专门审批流程

## Q4. 成本拆解—差异金字塔 (10分)

### 基础计算

预算总成本: 5,500×6.0×25.00+5,500×1.50=825,000+8,250=**833,250**
实际总成本: 7,000×4.6×26.80+7,000×1.20=863,296+8,400=**871,696**
(注:运费单独列,¥1.50/kg预算→¥1.20/kg实际)

预算用量: 5,500×6.0=33,000kg; 实际用量: 7,000×4.6=32,200kg
预算含运费单价: ¥25.00+¥1.50=¥26.50/kg
实际含运费单价: ¥26.80+¥1.20=¥28.00/kg

### L0总差异

总成本差异: 871,696-833,250=**+38,446**

或者按kg口径:
预算总kg成本: 33,000×(25.00+1.50)=874,500? 等等...
让我重新精确计算:

预算不含运费: 33,000×25.00=825,000
预算运费: 5,500×9.00=49,500? → 题目说1.50/kg→5,500×6.0×1.50=49,500
预算总计: 825,000+49,500=**874,500** ← 正确

实际不含运费: 32,200×26.80=862,960
实际运费: 7,000×5.52×1.20=46,368? → 32,200×1.20=38,640
实际总计: 862,960+38,640=**901,600**

L0 = 901,600-874,500 = **+27,100**

### 差异金字塔

L1产量效应(按预算单位):
标准用量(按实际产量): 7,000×6.0=42,000kg
预算用量: 33,000kg
L1 = (42,000-33,000)×(25.00+1.50) = 9,000×26.50 = **+238,500** (产量增→成本增,合理)

L2效率差异:
实际用量: 32,200kg
L2 = (32,200-42,000)×26.50 = -9,800×26.50 = **-259,700** (比标准少用料→节约!)

L3价格差异:
预算含运费单价: 26.50/kg
实际含运费单价: (862,960+38,640)/32,200=901,600/32,200=28.00/kg
L3 = (28.00-26.50)×32,200 = 1.50×32,200 = **+48,300** (涨价)

### 闭合验证

L1+L2+L3 = 238,500+(-259,700)+48,300 = **27,100** ✅
偏差 = |27,100-27,100|/874,500 = **0%** 完美闭合!

### 管理分析

- 产量效应+238,500: 合理增产导致的成本增长
- 效率差异-259,700: 单位用料从6.0→4.6kg(改善23%)→大幅节约!这是最关键发现
- 价格差异+48,300: 含运费单价从26.50→28.00(+5.7%)→涨价侵蚀

**主因**: 效率改善(-259,700)几乎抵消了产量增长(+238,500),成本超支主要来自涨价(+48,300)。这是典型的好消息(效率大幅提升)被坏消息(原料涨价)部分掩盖。

建议: ①锁定原料价格 ②推广新工艺到其他产品线 ③运费单独议价(1.20<1.50已节约)
