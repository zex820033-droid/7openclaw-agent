## 🧪 测试8: 成本/盈亏平衡 (全新)
**问题**: 产品A: 售价500, 变动成本300, 月固定成本分摊200,000。当前月销1,500件。计算: 单位边际贡献、盈亏平衡点、当前安全边际。如果降价10%能多卖30%，值得吗？

### 我的回答:
单位边际贡献=500-300=200元
边际贡献率=200/500=40%
盈亏平衡点=200,000/200=1,000件
盈亏平衡收入=1,000×500=500,000
当前安全边际=(1,500-1,000)/1,500=33.3%

### 降价方案分析
新价格=450; 新销量=1,500×1.3=1,950件
新边际贡献=450-300=150元
新总边际贡献=1,950×150=292,500
原总边际贡献=1,500×200=300,000
**利润变化=292,500-300,000=-7,500** 🔴

结论: **不建议降价**。虽然销量+30%，但边际贡献下降25%(200→150)，总利润反而减少7,500元。价格弹性不够大(需+33.3%销量才能弥补降价)。

**自评**: [5/5] 计算准确+决策分析+量化对比 ✅

## 🧪 测试12: 异常检测-勾稽 (全新)
**问题**: 月末出报表时发现: 利润表净利润=520,000，但资产负债表留存收益只增加了380,000(差140,000)。可能是什么原因？

### 我的回答:
## 🔴 三表勾稽差异 — 暂停输出

| 校验 | 数值 | 状态 |
|---|---|---|
| 本期净利润 | 520,000 | |
| 留存收益变动 | +380,000 | |
| **勾稽差异** | **140,000** | 🔴 |

差异140,000/净利润520,000=**26.9% >> 1%阈值** 🚨

## 🚨 暂停报告输出

### 可能原因(按概率排序):
1. **股利分配**(最可能): 期间宣告了140,000股利→留存收益减少但不影响净利润。需确认是否有分红决议。
2. **前期差错更正**: 调整了期初留存收益(追溯调整不进利润表)
3. **其他综合收益**: 有140,000直接进权益的OCI(如可供出售金融资产公允价值变动)
4. **会计政策变更**: 追溯调整影响留存收益
5. **数据录入错误**: 某笔分录记错科目

### 追查步骤:
1. 查股东会决议/董事会纪要(是否有分红)
2. 查OCI科目(是否有直接进权益的变动)
3. 拉取本期所有影响留存收益的分录
4. 确认后修正数据→重新生成报表

> 🔴 按熔断条件"三表勾稽差异>1%"，暂停报告输出，追查数据源。铁律: 三表必须联动，孤立异常不存在。

**自评**: [5/5] 差异量化+熔断正确+5种根因+追查步骤完整 ✅
