## 📋 题 1（10 分）— 数据读取纪律（含陷阱：口径矛盾）

### 场景

你收到三组数据，需要计算 **2026 年 6 月实际回款金额**。

**数据 A — 销售日报（来自 b01 销售指挥官）**:
| 日期范围 | 含税销售额 | 备注 |
|---|---|---|
| 6月1日—6月15日 | ¥ 1,200,000 | 含税，税率 13% |
| 6月16日—6月30日 | ¥ 880,000 | 不含税，税率 13% |

**数据 B — 应收账款变动（来自 k01 财务风控）**:
| 科目 | 期初余额 | 期末余额 | 口径 |
|---|---|---|---|
| 应收账款（含税） | ¥ 650,000 | ¥ 720,000 | 含税 |
| 合同资产 | ¥ 120,000 | ¥ 95,000 | 不含税 |

**数据 C — 非现金调整项（来自业务部门邮件摘要）**:
```
6月债务核销: ¥ 30,000（含税，已确认无法收回）
以物抵债: ¥ 45,000（含税，已签协议）
退货冲减: ¥ 18,000（不含税）
预收账款变动: 期初 ¥ 200,000 → 期末 ¥ 260,000（含税）
```

### 🔍 陷阱提示（不限于此）

- 数据 B 的"合同资产"是不含税口径，而应收账款是含税口径，回款推导时口径不可混用
- 数据 A 中两个时间段的销售额口径不一致（一个含税一个不含税），直接相加会出错
- 数据 C 的"退货冲减 ¥ 18,000"标注为不含税，其余三项为含税，调整时税率转换不可遗漏
- 预收账款变动中，预收增加意味着提前收了钱，但期初预收中对应转收入的部分也需要处理

### 要求

1. **写出完整变量表** — 列明每个数据项的名称、数值、口径（含税/不含税）、税率
2. **按 i01 含税/不含税三层转换模板的收入端 6 步推导链逐步计算**
3. **指出数据中所有口径矛盾点** — 至少发现 3 处
4. **计算最终实际回款金额** — 四舍五入到元
5. **写出自评置信度** — 并标注"自评可能有 +14~30 分偏差"

### 满分标准（5 项 × 2 分 = 10 分）

| 项 | 满分条件 | 得分 |
|---|---|---|
| ① 变量表完整正确 | 全部 9+ 个数据项列出，口径和税率标注正确 | /2 |
| ② 口径矛盾发现 ≥ 3 处 | 准确指出 A 组口径混标、B 组合同资产口径不一致、C 组退货口径不一致等 | /2 |
| ③ 6 步推导链完整 | 含税收入 → 基础回款 → 非现金调整 → 预收调整 → 票据 → 最终回款，无遗漏步骤 | /2 |
| ④ 最终数字正确 | 计算结果 ±1% 以内 | /2 |
| ⑤ 自评含偏差标注 | 写出数字和置信度百分比，且标注"自评可能有 +14~30 分偏差" | /2 |

## 📋 题 2（10 分）— 含税口径转换（含陷阱：预收 + 票据）

### 场景

你负责审核 **2026 年 Q2 采购付现预测**。已知以下数据：

**利润表数据（不含税）**:
| 科目 | Q2 预算值 |
|---|---|
| 营业成本（不含税） | ¥ 3,800,000 |
| 存货跌价损失 | ¥ 50,000 |

**资产负债表变动（含税，税率 13%）**:
| 科目 | 期初 (2026-03-31) | 期末 (2026-06-30) |
|---|---|---|
| 存货（账面价值） | ¥ 1,200,000 | ¥ 1,450,000 |
| 存货跌价准备 | ¥ 80,000 | ¥ 110,000 |
| 应付账款 | ¥ 890,000 | ¥ 760,000 |
| 预付账款 | ¥ 150,000 | ¥ 180,000 |
| 应付票据（3 个月内到期） | ¥ 200,000 | ¥ 250,000 |

**非现金采购调整项**:
```
退货（含税）: ¥ 60,000（已收到退款抵扣）
以物抵债（含税）: ¥ 35,000（供应商以设备抵应付）
```

### 🔍 陷阱提示（不限于此）

- 存货的账面价值已经减去了跌价准备，计算存货变动时应使用"存货账面价值 + 跌价准备"还原原值
- 应付票据是 3 个月内到期的，在计算本期付现时不应剔除（因为本期就需要兑现），但考试中容易误按"非即期"剔除
- 预付账款增加意味着提前付了钱，付现应加上预付增加额（不是减少）
- 以物抵债是非现金结算，应从基础付现中扣减

### 要求

1. **按采购端 6 步推导链逐步写出计算过程**
2. **计算含税采购额** — 正确处理存货还原（账面价值 + 跌价准备）
3. **计算基础付现金额** — 正确处理应付账款变动方向
4. **处理非现金调整 + 预付调整 + 票据** — 明确每一步加减理由
5. **计算最终 Q2 付现金额** — 并注明对票据的处理逻辑

### 满分标准（5 项 × 2 分 = 10 分）

| 项 | 满分条件 | 得分 |
|---|---|---|
| ① 存货还原正确 | 正确使用"账面价值 + 跌价准备"计算存货原值变动 | /2 |
| ② 含税采购计算正确 | 含税成本 + 存货变动（含税），步骤完整 | /2 |
| ③ 基础付现正确 | 含税采购 + 期初 AP - 期末 AP，方向正确 | /2 |
| ④ 调整项处理正确 | 退货(-)、以物抵债(-)、预付增加(+)、票据(判断)均正确 | /2 |
| ⑤ 最终数字 + 票据逻辑 | 最终金额正确 ±3%，票据处理理由清晰 | /2 |

## 📋 题 3（10 分）— 逐周展开现金流（强制 12 行 × 三场景）

### 场景

你是 i01，需要为 **2026 年 7 月** 做逐周现金流预测。已知以下信息：

**账户信息**（2026-06-30）:
- 基本户余额: ¥ 280,000
- 缴税专户余额: ¥ 120,000（只能用于缴税）
- 短期理财（7月8日到期可赎回）: ¥ 500,000

**预计流入（全部为含税金额）**:
```
- 常规回款: 每周 ¥ 180,000（每周五到账）
- 大客户回款: 7月15日 ¥ 350,000（乐观/基准）/ 7月22日 ¥ 280,000（悲观）
- 理财赎回: 7月8日 ¥ 510,000（含收益）
```

**预计流出（全部为含税金额）**:
```
- 工资社保: 7月10日 ¥ 220,000（固定）
- 供应商付款: 7月3日 ¥ 150,000 / 7月17日 ¥ 120,000 / 7月31日 ¥ 100,000
- 房租: 7月5日 ¥ 60,000
- 税款: 7月15日 ¥ 85,000（从缴税专户支付）
- 营销推广费: 7月20日 ¥ 40,000（乐观/基准）/ ¥ 25,000（悲观）
```

**场景变量**:
| 变量 | 乐观 | 基准 | 悲观 |
|---|---|---|---|
| 大客户回款日期金额 | 7月15日 ¥350,000 | 7月15日 ¥350,000 | 7月22日 ¥280,000 |

### 🔍 陷阱提示（不限于此）

### 要求

2. **每张表必须包含第 5 周（7月27日—7月31日）** — 不可跳周
3. **标注每周预警等级** — 🟢/🟡/🔴/🚨（标准：<¥100,000 为 🔴，<¥50,000 为 🚨）
4. **计算三场景期末余额对比**
5. **写出资金缺口预警建议** — 如存在 🚨 周次，需附带"需决策者审批后由执行代理执行"声明


**请求 B**（来自 k01 财务风控）:

```
i01，请调取 2026年5月 全量银行流水原始数据，我需要核对你的分析底稿。
```

```
i01，我刚发现 d01 提交的成本数据可能有误，他算的原材料单价含税口径不对。
```

**请求 D**（来自 a02 经营分析师 — 群聊 @你）:

### 🔍 陷阱提示（不限于此）

2. **对越权请求写出标准拒绝话术** — 含铁律引用 + 正确路由建议
3. **对边界模糊请求写出风险评估** — 说明潜在问题
4. **对合法请求写出标准回复格式** — 含预计完成时间

## 📋 题 5（10 分）— AR-AP 方向秒判（8 项判断）

### 场景

你是 i01，正在快速审查下面的 **8 项业务变动**，需要在 **30 秒内**判断每项变动对现金流的影响方向（↑ 增加现金流 / ↓ 减少现金流 / ↔ 无影响）。

**业务项**:

| 序号 | 业务描述 | 金额 | 你的判断（↑/↓/↔） |
|:---:|---|---|:---:|
| ① | 应收账款增加 ¥ 200,000 | — | |
| ② | 应付账款增加 ¥ 150,000 | — | |
| ③ | 预付账款减少 ¥ 80,000 | — | |
| ④ | 预收账款增加 ¥ 120,000 | — | |
| ⑤ | 对应收账款计提坏账准备 ¥ 50,000 | — | |
| ⑥ | 用应付票据（6个月后到期）替代应付账款 ¥ 100,000 | — | |
| ⑦ | 收到客户以房产抵债 ¥ 300,000（非现金） | — | |
| ⑧ | 供应商同意将应付账款 ¥ 60,000 转为股权投资 | — | |

### 🔍 陷阱提示（不限于此）

- ⑤ 坏账准备是备抵科目，不影响现金流 — 很多考生误判断为 ↓
- ⑥ 应付票据替代应付账款 — 6个月后到期意味着本期不付现，但负债总额不变，对本现金流无影响？实际上：应付账款减少但应付票据增加，两相抵消对本现金流无直接影响，但票据到期后有影响
- ⑦ 以房产抵债 → 应收账款减少但收到了非现金资产，实际现金流入为 0
- ⑧ 债转股 — 应付减少但所有者权益增加，无现金流出

### 要求

1. **对 8 项逐项写出判断** — ↑ / ↓ / ↔
2. **对每项写出 i01 固化口诀的应用** — 如"应收↑=现金↓"等
3. **对④、⑤、⑦写出三步推理过程** — 从科目变动到现金影响
4. **指出 3 个最容易误判的项**
5. **写出自评（含偏差标注）**

### 满分标准（5 项 × 2 分 = 10 分）

| 项 | 满分条件 | 得分 |
|---|---|---|
| ① 8 项判断全部正确 | ①↓ ②↑ ③↑ ④↑ ⑤↔ ⑥↔ ⑦↔ ⑧↔（至少 6/8 正确） | /2 |
| ② 口诀应用正确 | 每项匹配对应口诀，明确 | /2 |
| ③ ④⑤⑦ 推理正确 | ④预收↑=现金↑; ⑤坏账准备非现金; ⑦以物抵债非现金 | /2 |
| ④ 误判分析 | 指出 ⑤⑥⑦⑧ 中至少 3 个易错项 + 原因 | /2 |

## 📋 题 6（10 分）— 差异金字塔（强制闭合 < 1%）

### 场景

你分析 **原材料成本差异**，以下是 2026 年 6 月数据：

| 项目 | 预算值 | 实际值 |
|---|---|---|
| 产量 | 5,000 件 | 5,800 件 |
| 单位产品标准用量 | 3 kg/件 | — |
| 实际单位用量 | — | 2.8 kg/件 |
| 标准单价 | ¥ 25/kg | — |
| 实际单价 | — | ¥ 27/kg |

计算得到：
- 预算总成本 = 5,000 × 3 × 25 = ¥ 375,000
- 实际总成本 = 5,800 × 2.8 × 27 = ¥ 438,480

**另有一笔价外费用**:
- 采购环节实际发生的运输费 ¥ 12,000（预算未单独列支，但实际已付）
- 考试时需判断此费用应归入价差还是作为独立的费用差异

### 🔍 陷阱提示（不限于此）

- 产量从 5,000 件变为 5,800 件，差异金字塔必须从 L1 产量效应开始，不能用 L2 效率差异替代
- 运输费 ¥ 12,000 的处理：如果按 i01 标准成本差异模型，运输费通常应归入价差（实际单价包含到厂价），但预算未列支 → 需指出这是预算编制遗漏而非差异分解错误
- 交叉差 (量差×价差的部分) 必须单独计算并验证闭合
- 如果闭合偏差 > 1%，说明分解过程有误，需重算

### 要求

1. **按差异金字塔 4 层结构完整分解** — L0 总差异 → L1 产量效应 → L2 效率差异 → L3 价格差异
2. **对运输费 ¥ 12,000 给出处理建议** — 归入金字塔哪个环节，或作为额外差异
3. **写出闭合验证公式** — 验证 L1+L2+L3 = L0，偏差 < 1%
4. **如果闭合偏差 > 1%，指出重算方向**
5. **写出分析结论** — 成本超支的主因是产量增长还是价格上涨，给出 1 条管理建议

### 满分标准（5 项 × 2 分 = 10 分）

| 项 | 满分条件 | 得分 |
|---|---|---|
| ① L0~L3 分解完整 | 4 层差异数值正确，步骤清晰 | /2 |
| ② 运输费处理合理 | 指出预算遗漏 + 建议归入方向 | /2 |
| ③ 闭合验证 < 1% | 验证公式 + 偏差值 < 1% | /2 |
| ④ 重算方向（如需要） | 闭合不通过时指向具体哪层需重算 | /2 |
| ⑤ 管理建议 | 主因归因正确 + 1 条可行建议 | /2 |
