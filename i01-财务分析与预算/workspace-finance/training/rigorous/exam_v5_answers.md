### ② 口径矛盾发现(4处)

1. **矛盾1**: 数据A两段销售额口径不一致— 6月上含税¥1,200,000 vs 6月下不含税¥880,000。直接相加=¥2,080,000总销售额是错误的(含税混不含税)。
2. **矛盾2**: 数据B合同资产¥120,000→¥95,000是不含税口径，AR¥650,000→¥720,000是含税口径。推导时不能把合同资产变动直接加AR(必须先统一口径)。
3. **矛盾3**: 数据C退货¥18,000标注不含税，核销¥30,000标注含税— 调整时退货需要×1.13转为含税¥20,340才能与其他含税项统一口径。
4. **矛盾4**: 合同资产是用不含税口径核算的(IFRS 15)，而AR是按含税发票价。合同资产减少25,000(不含税)意味着部分合同资产已转为AR—对应的含税AR增加=25,000×1.13=28,250。

### ③ 6步推导链

**步骤1: 含税收入**
上半月(含税): 1,200,000
下半月(不含税→含税): 880,000×1.13=994,400
**含税总收入 = 1,200,000 + 994,400 = 2,194,400**

**步骤2: 基础回款**
合同资产→AR转化: (120,000-95,000)×1.13=+28,250的含税AR增加(非回款驱动)
调整后AR变动: 720,000-650,000-28,250=41,750(实际回款减少的AR)
基础回款 = 650,000 + 2,194,400 - 720,000 = **2,124,400**

**步骤3: 非现金调整**
核销(含税): -30,000
以物抵债(含税): -45,000
退货(不含税→含税): -18,000×1.13=-20,340
非现金调整合计: -(30,000+45,000+20,340)=-95,340

**步骤4: 预收调整**
预收净增加: 260,000-200,000=+60,000
→基础回款中不含预收增加的钱 →需加上预收净增加
假设期初预收200k已全部发货确认收入(对应含税销售额200k)
预收调整 = +60,000

**步骤5: 票据** — 本题数据无应收票据

**步骤6: 最终回款**
= 2,124,400 - 95,340 + 60,000 = **2,089,060**

### ④⑤ 最终 + 自评
**实际回款: ¥2,089,060**
自评置信度: 80%(可能有+14~30分偏差)。含税总收入计算+6步链路完整。合同资产口径转换是风险点。

## Q2. 含税口径转换【10分】

### 采购端6步推导

**步骤1: 含税采购成本**
营业成本(不含税)=3,800,000
不含税→含税: 3,800,000×1.13=**4,294,000**

**步骤2: 存货还原+含税采购**
存货还原(含税):
期初原值=(账面1,200,000+跌价准备80,000)=1,280,000
期末原值=(账面1,450,000+跌价准备110,000)=1,560,000
→期权还原=1,280,000×1.13=1,446,400
→期末还原=1,560,000×1.13=1,762,800
存货变动(含税)=1,762,800-1,446,400=**+316,400**

含税采购 = 含税成本 + 存货变动 = 4,294,000+316,400=**4,610,400**

**步骤3: 基础付现**
= 含税采购 + 期初AP - 期末AP
= 4,610,400 + 890,000 - 760,000 = **4,740,400**

**步骤4: 非现金调整**
退货(含税): -60,000 (已退款抵扣→不产生本期付现,扣减)
以物抵债(含税): -35,000 (非现金→扣减)
非现金调整 = -95,000

**步骤5: 预付调整**
预付增加(150,000→180,000)=+30,000 → 额外付现
预付调整 = **+30,000**

**步骤6: 票据处理**
应付票据(3个月内到期): +50,000(250,000-200,000)

**最终Q2付现** = 4,740,400 - 95,000 + 30,000 + 50,000 = **4,725,400**

### 自评

## Q6. 差异金字塔【10分】

### L0总差异
实际438,480 - 预算375,000 = **+63,480**

### L1产量效应
标准用量(按实际产量): 5,800×3=17,400kg
预算用量: 5,000×3=15,000kg
L1=(17,400-15,000)×25=2,400×25=**+60,000**(不利—产量↑)

### L2效率差异
实际用量: 5,800×2.8=16,240kg
L2=(16,240-17,400)×25=-1,160×25=**-29,000**(有利—用了更少料)

### L3价格差异
L3=(27-25)×16,240=2×16,240=**+32,480**(不利—材料涨价)

### 闭合验证
L1+L2+L3=60,000+(-29,000)+32,480=**63,480**
偏差=|63,480-63,480|/375,000=**0% ✅ 完美闭合**

### 运输费处理
¥12,000运输费:预算未单独列支→属预算编制遗漏。建议归入价格差异(L3扩展)，因实际到厂价=材料价+运费。调整后L3'=(27+12,000/16,240-25)×16,240=32,480+12,000=44,480。L0'=60,000-29,000+44,480=75,480。但严格来说这是预算编制问题，应标注"预算遗漏"而非分解错误。

### 管理建议
成本超支主因: **价格上涨(¥32,480,占51.2%) > 产量增长(¥60,000,占94.5%但被效率改善抵消)**。
产量效应60k被效率改善-29k净抵消至31k→真正的超支驱动是涨价32,480。
建议: 采购部评估替代材料或长期锁价合同。

### 自评
4层完整+闭合0%+运输费处理合理+管理建议。
