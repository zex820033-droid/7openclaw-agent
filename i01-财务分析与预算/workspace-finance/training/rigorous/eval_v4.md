### 教练命令 #2：【B区含税口径专项——6步推导链模板】

**问题**: Q4预收处理（混用银行承兑500k替代预收转收入320k）、Q5全链崩溃、Q6税额勾稽公式不完整。B区（含税口径转换，全卷最核心区块）得分率仅57%。

**教练命令**:

```
B区（Q4-Q6）答题必须使用「6步推导链模板」，每步强制标注(含税)/不含税/税率：

Q4 收入端模板：
  1. 含税收入 = 不含税收入 × (1+税率)
  2. 基础回款 = 期初AR(含税) + 含税收入 - 期末AR(含税)
  3. 非现金调整: 核销(-) + 以物抵债(-)
  4. 预收调整: + 预收增加 - 期初预收转收入
  5. 应收票据: 确认到期时间，非即期剔除
  6. 最终公式: 回款 = 步骤2 - 步骤3非现金调整 + 步骤4预收调整 (不含步骤5的银行承兑)

Q5 采购端模板：
  1. 含税成本 = 不含税成本 × (1+税率)
  2. 含税采购 = 含税成本 + 存货变动(含税)
  3. 基础付现 = 含税采购 + 期初AP(含税) - 期末AP(含税)
  4. 非现金调整: 退货(-若冲应付且未退款) + 应付票据(+新开票)
  5. 预付调整: - 预付增加(新付现金) 或 + 预付减少(前期已付)
  6. 最终付现 = 步骤3 - 步骤4非现金调整 + 步骤5预付调整

Q6 税额勾稽模板：
  实缴增值税 = 期初应交增值税 + 本期销项 - 本期进项 - 期末应交增值税
  实缴所得税 = 期初应交所得税 + 本期计提 - 期末应交所得税
  总实缴税费 = 实缴增值税 + 实缴所得税 + 附加税(增值税×12%)
  与CF"支付的各项税费"对比 → 残差归因（附加税/印花税/房产税/资本性进项）
```

**验收**: V4 B区(Q4-Q6)必须使用该模板，每步标注口径。任何一步缺失或口径未标注，该题扣50%分。
