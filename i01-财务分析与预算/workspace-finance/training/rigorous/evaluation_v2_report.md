## 🎓 教练命令：5条具体改进指令

> 以下指令按优先级排序。每条指令包含**具体行动**和**验收标准**。

---

### 指令 1: 含税/不含税口径——建立三层口径转换表 🔴

**问题**: Q5 中 280k 差异未完全消解，根因是含税收入、含税回款、含税采购之间的口径转换不熟。

**行动**:
1. 在 `training/` 下新建 `tax_caliber_drill.md`
2. 构造 10 组"从利润表科目→现金流科目"的含税/不含税转换题
3. 每组必须包含：收入(不含税)→收入(含税)→回款(含税)→回款验证，以及成本(不含税)→采购(含税)→付现(含税)→付现验证
4. 每组做完后自查：`Σ子差异 = 总差异`（误差 < 1%）

**验收标准**: 下一轮考试出现含税口径转换时，差异归因的未解释残差 < 5%。

**格式模板**:
```
## 口径转换题 #N
| 报表项目 | 不含税 | 税率 | 含税 |
|...计算链...
| 回款推导 | 期初AR + 含税收入 - 期末AR ± 预收变动 = 推导回款 |
| 差异消解 | 推导回款 vs 报表回款 = 差异 → 归因到具体科目 |
```

---

### 指令 2: 悲观场景——强制逐周逐科目展开 🟡

**问题**: Q7/Q9 悲观场景只做关键周估算，未逐周展开。现金流建模的核心纪律是"每个数字都可追溯到具体假设"。

**行动**:
1. 重新做 Q7 悲观场景——**逐周**计算 W1-W12，每行标注计算依据
2. 重新做 Q9 悲观+台风场景——**逐周**计算 W1-W12，叠加台风影响
3. 对每个场景标注：
   - 每周的预警等级（🟢🟡🔴🚨）
   - 最低现金头寸出现的**精确周**
   - 从当前周到最低周的**累计净流出**
4. 输出格式：三场景并排的逐周矩阵表

**验收标准**: 下一轮考试出现现金流预测时，三场景全部逐周展开，不出现"估算""约"等模糊词。

**警告**: 悲观场景不可以"太多了算不完"跳过——这恰恰是现金流建模最有价值的场景。

---

### 指令 3: 预警矩阵——建立"逐场景×逐周×逐等级"的三维标注 🟡

**问题**: Q7 只在基准场景标注了 W6 最低、W10 断裂，但没有对三场景的每周做预警等级标注。

**行动**:
1. 对 Q7 重做的三场景逐周表，在每周末尾增加一列 `预警等级`
2. 判定规则：
   - 🟢 安全：期末现金 > 2×月固定支出（1,960k）
   - 🟡 关注：期末现金 在 月固定~2×月固定之间（980k~1,960k）
   - 🔴 危险：期末现金 < 月固定支出（980k）
   - 🚨 紧急：期末现金 < 0 或 30天内缺口 > 月支出×50%
3. 对 Q9 重做的悲观+台风逐周表，同样标注
4. 统计每场景下各级预警出现的周数，作为风险画像

**验收标准**: 任何现金流预测输出，必须附带完整的预警矩阵。

---

### 指令 4: 应收/应付变动方向→现金流方向的直觉固化 🟡

**问题**: Q5 中 "实际AR↑320k vs 预算AR↑400k→实际多占-80k(实际更好?)" 显示方向直觉不稳。

**行动**:
1. 背诵并内化两条铁律：
   - **应收增加 = 现金减少**（客户欠你的钱多了 → 你实际收到的钱少了）
   - **应付增加 = 现金增加**（你欠供应商的钱多了 → 你实际付出的钱少了）
2. 在 `training/` 下新建 `ar_ap_direction_drill.md`，构造 20 组快速判断题
3. 每题格式：
   ```
   | 科目 | 期初 | 期末 | 变动 | 对现金影响(±) |
   | 应收 | 100 | 150 | +50 | _____ |
   | 应付 | 80  | 60  | -20 | _____ |
   ```
4. 做到秒答不出错为止

**验收标准**: 在任何涉及应收应付的题目中，方向判断零错误。

---

### 指令 5: 多层级差异分解——总差异=Σ子差异的强制闭合 🔵

**问题**: Q12 的差异金字塔 L2 产量效应未单独结构化为一个明确的子项。

**行动**:
1. 对 Q12 重做完整的差异金字塔，强制要求 `Σ子差异 = 总差异`（允许 < 1% 截断误差）
2. 层级结构：
   - **L0 总差异**: 实际总成本 - 预算总成本
   - **L1 产量效应**: (实际用量 - 预算用量) × 预算单价
   - **L2 配方价格效应**: (实际配方单价 - 预算配方单价) × 实际用量
   - **L3 料比效应**: (实际料比 - 预算料比) × 实际产量 × 预算配方单价 / 1000
   - **L4 交叉项**: L0 - (L1+L2+L3)
3. 每层子差异都要有一个**业务含义标签**（有利/不利 + 一句话根因）
4. 推广到所有成本差异分析——以后每道量价分离题都必须走到金字塔底层

**验收标准**: 任何量价分离题输出的差异金字塔，顶层总差异与底层子差异之和的偏差 < 1%。
