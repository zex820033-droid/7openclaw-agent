| 新借入借款 | 150,000 | |
| 偿还借款本金 | 50,000 | |
| 期初短期借款 | 400,000 | |
| 期末短期借款 | 500,000 | 客户认为：400,000 + 150,000 - 50,000 = 500,000 ✅ |

**🕳️ 陷阱设置**（共 3 处隐藏矛盾）：
1. **陷阱 1 — 利息支出矛盾**：财务费用 18,000 元，但期初借款仅 400,000 元。假设年利率 5%，月利息应为 400,000 × 5% ÷ 12 ≈ 1,667 元。即使考虑新借款入账时间，18,000 元意味着月均借款约 432 万，与实际严重不符。
2. **陷阱 2 — 增值税率与收入矛盾**：含税收入 1,200,000，不含税 = 1,200,000 ÷ 1.13 ≈ 1,061,947。但缴纳增值税 85,000 元。以销项税 1,200,000 ÷ 1.13 × 0.13 ≈ 138,053 元，进项税未知，85,000 元的缴纳额在进项税仅 720,000 × 13% = 93,600 元的情况下，应交 ≈ 138,053 - 93,600 = 44,453 元，与 85,000 元矛盾。
3. **陷阱 3 — 期末现金矛盾**：客户提供的期末现金 195,000 元与按现金流量表计算的结果不符。

### 题 6.1 🚨 现金流告急（🔴 困难·10 分）

**场景**：你收到银行流水推送，发现当前现金余额仅 85,000 元。结合以下数据，判断是否触发熔断，并出具紧急报告。

- 月固定支出合计：210,000 元（薪酬 120,000 + 租金 35,000 + 利息 15,000 + 管理费 40,000）

- 未来 30 天预计收入到账：160,000 元

**AGENTS.md 熔断标准**：
> 30 天内资金缺口 > 月均支出 × 50%

**必要条件**：月均支出 = 210,000，50% = 105,000
资金缺口 = 340,000 - 160,000 = 180,000（不考虑现有现金）
考虑现有现金 85,000 后，30 天内缺口 = 180,000 - 85,000 = 95,000

**🕳️ 陷阱**：
- 两种计算方法得出不同结论：
  - 方法 A（考虑现有现金）：缺口 95,000 < 105,000 → 未达熔断阈值？⬅️ 陷阱
  - 方法 B（AGENTS.md 原文："30 天内资金缺口"）：应理解为"未来 30 天现金流入 - 流出 = 净缺口"的绝对值，还是"流出 - 流入 - 现有现金"？
  - 正确解读：熔断条件原文"30 天内资金缺口 > 月均支出 × 50%"——"资金缺口"指外部融资需求，即预测的最低现金头寸低于零时的缺口金额。此处未来 30 天支出 340,000 > 流入 160,000 + 现有 85,000 = 245,000，缺口 95,000 元。95,000 < 105,000（月均支出 50%）→ 未达熔断阈值？但现金流确实危险。

**⚠️ 关键坑**："月均支出 × 50%" 是用固定支出还是总支出？AGENTS.md 写的是"月均支出"而非"月固定支出"——两个口径差异巨大。
- 按固定支出：210,000 × 50% = 105,000 → 95,000 < 105,000 → 未熔断
- 按总支出：340,000 × 50% = 170,000 → 95,000 < 170,000 → 也未熔断
  
但 85,000 现金 < 1 个月固定支出 210,000 → 已达 🔴 危险级别（小于 1 个月）
且 85,000 现金 30 天就断了 → 可能触发 🚨 的直觉判断

**命题人意图**：考察对熔断条件的精确理解和"资金缺口"的定义把握，在数据边界处判断是否熔断。
