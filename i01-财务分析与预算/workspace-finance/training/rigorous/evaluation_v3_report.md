## 📢 给 i01 的考前提醒

> i01，V3 试卷已经就位。以下是你需要知道的：

### 关键变化

1. **含税/不含税是 V3 的核心战场** — A区占 25% 总分。如果你不能在 Q1-Q3 拿到 ≥ 27 分，Stage 4 几乎不可能。
2. **闭合验证是硬纪律** — Q8/Q9 强制要求 Σ子差异=总差异 < 1%。不做的直接后果是最多得 60% 分（-4分起扣）。
3. **陷阱密度翻倍** — V2 是 18 陷阱/12 题 = 1.5/题，V3 是 35/12 = 2.9/题。几乎每道题都有 2-3 个陷阱。
4. **悲观场景不能"估算"** — Q4 明确规定"不允许合并计算或关键周估算"，违规扣 4 分。
5. **方向判断题不能出错** — Q7 的 10 组场景，任何一个方向错误 = 丢 1 分。这是"肌肉记忆"题。

### V2 中最需要警惕的习惯

- "做了大概方向对" → V3 要求精确到残差 < 1%
- "考点都覆盖了" → V3 要求每道题的**每个评分项都精确满足**
- "含税口径提一嘴就行" → V3 要求逐层展开含税/不含税转换表
