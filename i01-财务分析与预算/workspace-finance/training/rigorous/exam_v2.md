## Q10. 原料成本量差/价差分解 【10分】

**题干**：

虾类加工企业 2026 年 5 月原料成本较预算超支。请完成量差与价差的完整分解，判断何者为主因。

### 基础数据

| 指标 | 预算 | 实际 |
|---|---|---|
| 原料-鲜活虾用量 (kg) | 15,000 | 13,200 |
| 原料-鲜活虾单价 (¥/kg) | 52.00 | 58.50 |
| 原料-鲜活虾总成本 (¥) | 780,000 | 772,200 |
| 原料-辅料（调味料）总成本 (¥) | 120,000 | 138,000 |
| 产成品产量 (kg) | 12,000 | 9,500 |

> 🪤 **陷阱 1**：鲜活虾总成本看似下降了 ¥7,800，但这是量价反向运动的结果——不要被总成本表象迷惑  
> 🪤 **陷阱 2**：辅料成本的数据没有用量的单价，需要用其他方式分解

---

**满分标准（10分）**：

- [ ] **2分**：鲜活虾量差计算：
  - 量差 = (实际用量 - 预算用量) × 预算单价
  - = (13,200 - 15,000) × ¥52.00 = -¥93,600（有利差异，用量减少）
- [ ] **2分**：鲜活虾价差计算：
  - 价差 = (实际单价 - 预算单价) × 实际用量
  - = (¥58.50 - ¥52.00) × 13,200 = ¥85,800（不利差异，单价上涨）
- [ ] **1分**：交叉验证：量差 -¥93,600 + 价差 ¥85,800 = -¥7,800，与总成本差异 ¥772,200 - ¥780,000 = -¥7,800 一致 ✓
- [ ] **1分**：🪤 **陷阱洞察**：总成本下降 ¥7,800 是表象——实际是 ¥93,600 的用量节省被 ¥85,800 的涨价吃掉，净节省仅 ¥7,800。结论：**单价上涨是核心压力因素**
- [ ] **1分**：辅料成本分解——缺少用量和单价数据，无法做标准量价分离。此时应：
  - 请求补充辅料用量×单价明细
  - 或采用"单位产成品辅料成本"替代指标
  - 预算：¥120,000 / 12,000kg = ¥10.00/kg产成品
  - 实际：¥138,000 / 9,500kg = ¥14.53/kg产成品（+45.3%）⚠️
- [ ] **1分**：🪤 **陷阱识别**：辅料单位成本暴涨 45.3% 比鲜活虾涨价 12.5% 严重得多，但因为绝对金额小（¥18,000 vs ¥85,800）容易被忽略——应重点追查辅料异常
- [ ] **1分**：单位产成品原料成本的整体分析：
  - 预算：¥780,000 / 12,000kg = ¥65.00/kg
  - 实际：¥772,200 / 9,500kg = ¥81.28/kg（+25.1%）
  - 原料成本率从预算的某个%上升到实际的更高%
- [ ] **1分**：给出结论优先级：①辅料单位成本异常（+45.3%）→追查；②鲜活虾单价上涨（+12.5%）→关注趋势；③用量减少可能是产销量下降的结果而非效率提升→审慎解读

**扣分项**：

- 只看总成本差异不做量价分解：-3分
- 量差/价差公式用错（交叉项归属错误）：-2分
- 🪤 被总成本下降表象误导，得出"成本控制良好"错误结论：-3分
- 忽略辅料单位成本异常：-2分
- 未请求缺失数据或给出替代分析方法：-1分

## Q11. 人工成本效率差/工资率差分解 【10分】

**题干**：

加工车间 2026 年 5 月直接人工成本差异分析。请完成效率差异和工资率差异分解，并判断生产效率变化。

### 基础数据

| 指标 | 预算 | 实际 |
|---|---|---|
| 直接人工总工时 (h) | 4,800 | 4,180 |
| 直接人工总成本 (¥) | 168,000 | 158,840 |
| 平均小时工资率 (¥/h) | 35.00 | 38.00 |
| 产成品产量 (kg) | 12,000 | 9,500 |
| 单位产品工时 (h/kg) | 0.40 | 0.44 |

> 🪤 **陷阱**：总成本下降了 ¥9,160，总工时也下降了 620 小时——看似效率提升。但单位产品工时从 0.40 上升到 0.44，效率实际上是下降的。这是一个典型的"产量萎缩掩盖效率恶化"陷阱。

---

**满分标准（10分）**：

- [ ] **2分**：效率差异（量差）计算：
  - 效率差 = (实际工时 - 标准工时) × 标准工资率
  - 标准工时 = 实际产量 × 标准单位工时 = 9,500 × 0.40 = 3,800 h
  - = (4,180 - 3,800) × ¥35.00 = +¥13,300（不利差异，多耗380工时）
- [ ] **2分**：工资率差异（价差）计算：
  - 工资率差 = (实际工资率 - 标准工资率) × 实际工时
  - = (¥38.00 - ¥35.00) × 4,180 = +¥12,540（不利差异，工资率上升）
- [ ] **1分**：交叉验证：效率差 +¥13,300 + 工资率差 +¥12,540 = +¥25,840 总不利差异
  - 但总成本实际比预算低 ¥9,160（¥158,840 - ¥168,000 = -¥9,160）
  - 为什么？→ 产量差异效应：预算总工时 4,800h × ¥35 = ¥168,000 是基于 12,000kg 的产量
  - 如果按实际产量调整预算：3,800h × ¥35 = ¥133,000
  - 调整后差异 = ¥158,840 - ¥133,000 = ¥25,840 → 与量价分解一致 ✓
- [ ] **2分**：🪤 **陷阱穿透**：
  - 表面看：总工时 -620h，总成本 -¥9,160 → "效率不错"
  - 实际上：效率差 +¥13,300（多耗 380h）+ 工资率差 +¥12,540 = 总恶化 ¥25,840
  - **关键洞察**：产量下降 2,500kg（-20.8%）掩盖了生产效率的恶化。如果产量维持在 12,000kg，人工成本将达 ¥158,840 / 9,500 × 12,000 = ¥200,640，暴增 ¥32,640
- [ ] **1分**：单位产品工时从 0.40 → 0.44（+10%）的原因推测：产量下降导致固定人工无法等比减少/工人熟练度不足/设备故障增加等待时间
- [ ] **1分**：工资率从 ¥35 → ¥38（+8.6%）的原因推测：加班费增加/临时工比例变化/工资调整
- [ ] **1分**：给出改进建议：关注单位工时管控、分析加班原因、评估人员配置弹性

**扣分项**：

- 使用实际产量调整前预算直接比较得出错误结论：-3分
- 未识别"产量萎缩掩盖效率恶化"陷阱：-3分
- 效率差和工资率差公式用错：-2分
- 未做交叉验证（decomposition vs total variance reconciliation）：-1分
- 无改进建议：-1分

## Q12. 多因素成本差异——含配方替代陷阱 【10分】

**题干**：

虾饲料配方在 2026 年 5 月进行了调整（用豆粕替代部分鱼粉以降成本）。请完成多因素成本差异分解，特别注意配方替代效应。

> 🪤 **本题含 2 个陷阱**

### 配方数据

#### 预算配方（每吨饲料）

### 补充数据

| 指标 | 预算 | 实际 |
|---|---|---|
| 饲料总消耗量 (吨) | 380 | 345 |
| 饲料总成本 (¥) | 2,236,300 | 2,104,155 |

| 料比 (饲料kg/虾kg) | 3.17 | 3.63 |

> 🪤 **陷阱 1**：配方单价从 ¥5,885/吨 上升到 ¥6,099/吨（+3.6%），看似涨价——但鱼粉用量从 250kg→180kg（-28%），单价却从 ¥8.50→¥9.20（+8.2%），配方替代是否划算？  
> 🪤 **陷阱 2**：饲料总成本下降了 ¥132,145，但产量下降更厉害（-20.8%），料比从 3.17 恶化到 3.63（+14.5%）——这是更大的问题

**满分标准（10分）**：

- [ ] **1分**：配方成本差异（配方本身是否变贵了）：
  - 按预算配方单价计算的实际用量成本：345吨 × ¥5,885 = ¥2,030,325
  - 与实际成本 ¥2,104,155 的差 = +¥73,830（配方变贵了）
  - 进一步分解：配方单价差 = (¥6,099 - ¥5,885) × 345 = ¥73,830
  
- [ ] **2分**：产量差异（用量节省）：
  - 预算总成本 ¥2,236,300 - 调整后预算（实际产量×预算单价×预算料比）
  - 调整后预算 = 95,000kg × (3,800/120,000) × ¥5,885... 
  - 更清晰方法：量差 = (345 - 380) × ¥5,885 = -¥205,975（有利，少用35吨）
  - 但少用是因为产量下降 → 需进一步分解

- [ ] **2分**：🪤 **配方替代效应深度分解**：
  - 豆粕替代鱼粉的成本影响：
    - 鱼粉：预算 250kg × ¥8.50 = ¥2,125 → 实际 180kg × ¥9.20 = ¥1,656，节约 ¥469/t
    - 豆粕：预算 300kg × ¥3.80 = ¥1,140 → 实际 400kg × ¥4.50 = ¥1,800，增加 ¥660/t
    - 仅看这两项：替代净效果 = -¥469 + ¥660 = +¥191/t（替代反而更贵！）
    - 因为鱼粉涨价 ¥0.70/kg × 180kg = ¥126，豆粕涨价 ¥0.70/kg × 400kg = ¥280
    - + 鱼粉用量减少 70kg × ¥8.50 = -¥595，但豆粕用量增加 100kg × ¥3.80 = +¥380
    - 交叉项：减少鱼粉(70kg)按新老价差计算 → 过于复杂
    
- [ ] **2分**：料比恶化是最大黑洞：
  - 料比从 3.17 → 3.63（+14.5%）
  - 如果料比维持 3.17，实际产量 95,000kg 只需要 301 吨饲料
  - 实际用了 345 吨，多用了 44 吨 × ¥6,099 = ¥268,356 的额外成本
  - **料比恶化造成的损失远超配方替代的微小差异**
  - 🪤 这可能是豆粕替代鱼粉的副作用——饲料转化率下降

- [ ] **1分**：完整差异金字塔：
  - L1 总差异：¥2,104,155 - ¥2,236,300 = -¥132,145
  - L2 产量效应：(345-380) × ¥5,885 = -¥205,975
  - L2 配方价格效应：(¥6,099-¥5,885) × 345 = +¥73,830
  - L3 料比效应：如果料比不变应耗 301t，(345-301) × ¥6,099 = +¥268,356
  - L3 产量效应（纯）：(301-380) × ¥5,885 = -¥464,915
  - 总和 = -¥464,915 + ¥268,356 + ¥73,830 + 交叉项调整...

- [ ] **1分**：结论正确：①料比恶化是最大症结，成本影响远超配方替代；②豆粕替代鱼粉可能牺牲了饲料转化率；③如确认为替代副作用，需重新评估配方替代方案
- [ ] **1分**：给出具体建议：料比回升根因分析（配方/养殖管理/虾苗质量/病害）、配方替代的ROI重新测算
