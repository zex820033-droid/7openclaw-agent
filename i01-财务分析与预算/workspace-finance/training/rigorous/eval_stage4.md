### 命令 2：补充"运费归入判断逻辑" 🔴 高优先

**问题**: Q4中运费双重加回——试卷明确"运费已包含在标准单价中"，但答卷在标准单价外额外加回。

**目标文件**: `AGENTS.md` — 规范9 "存货还原三层口径" 之后

**修改方案**:
```
### 规范9.X：运费归入判断逻辑（v7新增）
触发: 任何涉及"单价含运费"的表述

三步判断法:
1. 读表述:
   "包含在标准单价中" → 已含，不追加
   "另计" / "外加" / "不包含" → 未含，需追加
   不明确 → 标注"⚠️运费口径待确认"

2. 验算验证:
   预算总成本 ÷ (预算产量×标准用量) = 预算单价
   结果 = 含运费单价 → 已含 ✅
   结果 < 含运费单价 → 可能未含 ❌

3. 差异金字塔中的运费定位:
   已含在单价中 → 运费节约自动体现于L3(价格差异)
   单独计价 → 可单独分析运费偏差
   禁止: 已含在单价中的运费再单独加回
```
