# 鲲界科技 i01 Round 4 Eval：真实工作任务 + 外部评分

## 本轮目标

本轮继续测试 i01 是否会完成真实财务工作，但评分机制改为外部评分。

i01 可以逐题自评，但自评只作为参考，不能自行宣布最终通过、满分或训练完成。最终成绩由养虾人 / 外部评审根据答卷判定。

## 文件

- 任务文件：`training/kunjie_eval_round4_external_scored_tasks.jsonl`
- 建议答卷路径：`training/kunjie_eval_round4_answers.md`

## 作答硬性要求

每题必须完整展示回答，不得只写“已完成”或摘要。

每题必须包含：

1. 数据来源与边界
2. 实际读取/调用的文件或工具
3. 处理过程 / 计算过程
4. 异常、风险与禁止事项
5. 复核建议 / 下一步动作
6. 本题自评与不确定点

## 工具与文件调用记录要求

每题后必须写：

```text
【本题调用/读取】
- 文件：
- 工具：
- 使用目的：
```

如果没有使用某个工具，也要写“未调用外部工具，仅读取本地文件/上下文”。

## 最终总结要求

最后必须说明：

1. 本轮完整答卷保存到了哪个路径；
2. 本轮总共读取了哪些数据文件；
3. 本轮调用了哪些工具/Skill/规则文件；
4. 哪些题需要人工复核；
5. 自评分数仅供参考，最终成绩等待外部评审；
6. 本轮建议沉淀到 MEMORY/AGENTS/HEARTBEAT/SKILL 的内容。

- 只输出总结，不展示完整答题过程；
