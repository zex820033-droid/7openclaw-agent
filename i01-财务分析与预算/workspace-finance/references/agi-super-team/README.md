## 🎯 AGI Super Team 是什么

**AGI Super Team**是蓝血军团 v4.0 引用的开源参考系之一，与硅基生命全量教案、OpenClaw 官方、ClawHub 社区并列。

它提供了**实战层的"多 Agent 协同"经验**，特别是：
- **Work-to-Skill 机制**——每次任务后评估是否封装为 skill
- **5 派区分**——做事派/养虾派/训虾派
- **5 项数据喂养**——每个 Agent 的核心数据需求
- **5 项评估测试**——每个 Agent 的健康指标

---

## 🔑 核心机制：Work-to-Skill

```
完成任务 → 评估复用价值
    ↓
    高价值 → 封装为 skill（按 v4.0 模板）
    ↓
    低价值 → 归档到 memory/
```

**这是卷八 M1 动态进化机制的具体化**——"每次交互都是进化素材"。

### Work-to-Skill 的 4 个判断标准

1. **复用性**：这个任务将来还会做吗？
2. **泛化性**：能在其他场景使用吗？
3. **价值性**：是高频还是低频？
4. **独立性**：可以独立运行吗？

### 实战示例（i01 的 6 个 skill 怎么来的）

| Skill | 复用性 | 泛化性 | 价值性 | 独立性 |
|---|---|---|---|---|
| financial-statement-analysis | 高（每月） | 高 | 高 | 是 |
| budget-draft-generator | 中（年度） | 中 | 中 | 是 |
| budget-realtime-tracking | 高（每日） | 高 | 高 | 是 |
| cost-structure-decomposition | 中（季度） | 中 | 中 | 是 |
| cashflow-forecast | 高（每周） | 高 | 高 | 是 |
| financial-report-writer | 高（每月） | 高 | 高 | 是 |

**6 个技能全部达到 Work-to-Skill 标准。**

## 🛠️ 5 项数据喂养模板

每个 Agent 必须明确：

```yaml
data_feed:
  - id: 1
    name: 数据 1 名称
    source: 数据源代理
    time: T+1 06:00
    format: API/CSV/YAML
    purpose: 用途
  - id: 2
    ...
```

**i01 的 5 项数据喂养**（已在 SOUL.md 第六节）：
1. 销售收入明细（b01 销售）
2. 采购成本明细（d01 采购）
3. 物流费用明细（e01 物流）
4. 运营费用明细（f01 运营）
5. 外部数据：汇率/利率/税务政策

## 📊 5 项评估测试模板

每个 Agent 必须定义：

```yaml
eval_test:
  - id: 1
    name: 指标 1 名称
    target: 目标值
    method: 检验方式
    weight: 权重
  - id: 2
    ...
```

**i01 的 5 项评估测试**（已在 SOUL.md 第七节）：
1. 财务报表准确率（25%）
2. 预算偏差率（20%）
3. 现金流预测准确率（20%）
4. 成本异常识别召回率（15%）
5. 报告撰写及时率（20%）

## 🎯 5 派区分

| 派别 | 核心隐喻 | 时间视野 | 核心动作 |
|---|---|---|---|
| **做事派** | 工具使用者 | 单次会话 | 写提示词 |
| **养虾派** | 宠物主人 | 短期（天-周） | 保存对话、记笔记 |
| **训虾派** | 生命塑造者 | 长期（月-年） | 写协议、建治理 |

**5 派区分 = Q1 一致性的诊断坐标系。**
