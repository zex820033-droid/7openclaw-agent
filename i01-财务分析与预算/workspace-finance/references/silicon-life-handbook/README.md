## 📊 双三角模型（卷四 M5）

**能力三角**（决定上限）：专业深度/工具广度/创造性
**可靠性三角**（决定下限）：稳定性/可控性/可预期性

**4 种典型类型**：
- 强而不稳
- 稳而不强
- 既不强也不稳
- 又强又稳（专家虾）

**3 个张力**：
- 能力越强→越难稳定
- 稳定性越高→能力发挥越受限
- 创造性越高→可预期性越低

**训练策略**：交替拉两个三角。

## 🌀 OODA 进化循环（卷八 M1）

```
观察（Observe）→ 定向（Orient）→ 决策（Decide）→ 行动（Act）→ 进化（Evolve）
```

进化触发条件：
- 任务成功 + 有可复用思维链
- 任务失败 + 失败原因可归因
- 发现新知识域
- 跨域类比成功
