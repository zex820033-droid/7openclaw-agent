## 🛠️ Work-to-Skill 机制（核心创新）

> **每次完成任务后评估是否值得封装为 skill**

```
完成任务 → 评估复用价值
    ↓
    高价值 → 封装为 skill（按 v4.0 模板）
    ↓
    低价值 → 归档到 memory/
```

**这是卷八 M1 动态进化机制的具体化**——"每次交互都是进化素材"。
