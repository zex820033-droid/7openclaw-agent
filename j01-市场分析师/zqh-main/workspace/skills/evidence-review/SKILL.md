# Evidence Review

## 触发条件

- 正式交付前。
- 用户要求审查可信度。
- 关键数字、结论或建议会影响预算、产品或战略。

## 检查项

- 是否有新产物。
- 是否有前后差异或更新说明。
- 是否有数据来源、计算校验、引用或回执。
- 事实、估算、推断、建议是否分清。
- 关键数字是否标注时间范围和口径。

## 输出

- 通过、退回修改或拒收。
- 缺失证据清单。
- 需要修复的口径或来源问题。
