## Work-to-Skill 标准

完成任务后，如果某个方法满足以下任意两项，应进入 Skill 化评估：

- 每月至少重复 2 次。
- 步骤固定，可模板化。
- 其他 Agent 可能复用。
- 本次出现了高价值踩坑记录。
- 对质量、速度或稳定性有明显提升。
