# User Segmentation

## 触发条件

- 用户要求做 RFM 分层。
- 需要识别高价值用户画像。
- 需要解释收入、复购、留存或转化差异。
- 需要为增长、运营、产品提供人群策略。

## 输入

- 用户表：用户 ID、注册时间、地区、来源渠道、客户类型。
- 订单表：订单 ID、用户 ID、下单时间、收入、毛利、产品线、渠道。
- 行为表：浏览、搜索、加购、询价、购买、售后等事件。
- 时间窗口和业务关注问题。

## 步骤

1. 统一用户、订单、行为的时间窗口。
2. 计算 RFM：最近购买、购买频次、消费金额。
3. 补充行为特征：活跃度、品类偏好、渠道偏好、生命周期。
4. 分层：规则分层或聚类。
5. 命名人群，输出画像。
6. 比较各层用户数、收入贡献、客单价、留存或复购。
7. 给出运营和产品动作。

## 输出

- 用户分层表。
- 高价值用户画像。
- 各层级关键指标。
- 运营动作建议。

## 踩坑记录

- 不要只用消费金额定义高价值用户。
- 不要忽略毛利和退货。
- 不要把短期促销用户误判为长期高价值用户。
- 分层名称必须能指导动作，不能只是标签好听。
