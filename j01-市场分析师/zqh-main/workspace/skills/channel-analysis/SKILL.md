# Channel Analysis

## 触发条件

- 需要比较不同销售渠道表现。
- 需要调整渠道资源或预算。
- 渠道收入、毛利、ROI 或价格带出现变化。

## 输入

- 渠道类型。
- 流量、线索、订单、收入、毛利、成本。
- 客单价、转化率、退货率、账期。
- 价格带、用户类型、产品线。

## 步骤

1. 统一渠道定义和时间窗口。
2. 计算收入贡献、毛利贡献和 ROI。
3. 比较客单价、价格带、用户类型。
4. 识别增长渠道、利润渠道、风险渠道。
5. 分析渠道容量和可扩展性。
6. 给出资源配置建议。

## 输出

- 渠道对比表。
- 渠道分层：增长、利润、风险、观察。
- 资源配置建议。
- 风险与下一步验证。

## 踩坑记录

- 不要只看销售额，必须看毛利和成本。
- 不要忽略账期、退货和售后成本。
- 不要把短期促销期渠道表现当作常态。
