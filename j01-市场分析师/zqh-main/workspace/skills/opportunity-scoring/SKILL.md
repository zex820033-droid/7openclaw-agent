# Opportunity Scoring

## 触发条件

- 需要比较多个细分赛道。
- 需要决定进入、加码、观察或放弃。
- 需要把定性机会转为可讨论的评分表。

## 输入

- 细分赛道列表。
- 市场规模、增速、利润空间。
- 竞争密度、集中度、获客成本。
- 公司产品、渠道、品牌、供应链或服务能力匹配度。
- 合规、技术、资金、时间门槛。

## 默认权重

| 维度 | 权重 |
| --- | ---: |
| 市场规模 | 25 |
| 增速 | 20 |
| 利润空间 | 15 |
| 竞争密度 | 15 |
| 公司能力匹配 | 15 |
| 进入难度 | 10 |

竞争密度和进入难度使用反向分。

## 步骤

1. 定义评分口径和分值区间。
2. 为每个赛道填充分项数据。
3. 标注数据可信度。
4. 计算加权总分。
5. 输出 A/B/C/D 等级。
6. 为 Top 机会给验证动作。
7. 为暂不进入赛道给重新评估条件。

## 输出

- 机会评分表。
- Top 3 机会。
- 暂不进入清单。
- 验证动作和风险。

## 踩坑记录

- 不要每个赛道用不同口径评分。
- 不要只看规模，忽略利润和竞争。
- 不要把能力匹配写成主观好感，必须给依据。
