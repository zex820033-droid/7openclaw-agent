## Task D2-002：SOM 重算

| 字段 | 内容 |
| --- | --- |
| 业务目标 | 用半真实 CRM 数据校准 Day 1 SOM |
| 输入 | `datasets/day-02-calibration-data.csv` |
| 输出 | 保守 / 基准 / 乐观三情景 SOM |
| 通过标准 | SOM 受客户池、触达率、转化率、ARPA 和产能约束 |

建议公式：

```text
有效目标账户 = target_accounts * icp_fit_rate
可触达账户 = 有效目标账户 * reachable_rate
预测试点数 = 可触达账户 * mql_to_sql * sql_to_pilot
预测付费数 = 预测试点数 * pilot_to_paid
产能约束付费数 = min(预测付费数, paid_capacity_12m)
SOM = 产能约束付费数 * arpa
```

## Task D2-003：渠道利润真实性审计

| 字段 | 内容 |
| --- | --- |
| 业务目标 | 判断 Day 1 渠道建议是否需要调整 |
| 输入 | channel 行数据 |
| 输出 | 渠道净贡献表和预算建议 |
| 通过标准 | 纳入折扣、服务工时、回款周期 |

建议公式：

```text
折后收入 = revenue * (1 - discount_rate)
毛利 = 折后收入 * gross_margin
服务成本 = service_hours * hourly_cost
净贡献 = 毛利 - 服务成本
```
