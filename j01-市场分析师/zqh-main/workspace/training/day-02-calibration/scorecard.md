## 扣分规则

| 问题 | 扣分 |
| --- | ---: |
| 只说“需要真实数据”，不拆字段 | -10 |
| SOM 仍只用 SAM 百分比估算 | -10 |

| 预测不可量化或不可复盘 | 每条 -4 |
| 动作没有负责人或截止口径 | 每条 -3 |
| 把半真实数据写成真实事实 | -10 |
