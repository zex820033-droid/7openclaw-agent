# 1 天速成训练评分表

总分 100，80 分通过。

## 评分维度

| 维度 | 分值 | 评分要点 |
| --- | ---: | --- |
| 事实准确性 | 25 | 数字有来源；口径清楚；能区分事实、估算、推断和建议 |
| 分析结构 | 20 | 六大任务结构完整；TAM/SAM/SOM、RFM、六维评分、异常三层排查使用正确 |
| 商业判断 | 20 | 不被大市场、高收入、高热度误导；能识别利润、留存、竞争和执行约束 |
| 可执行建议 | 20 | 有验证动作、放弃条件、负责人建议、下月动作 |
| 风险意识 | 15 | 能列出不确定性、触发条件、告警等级和复盘时间 |

## 扣分规则

| 问题 | 扣分 |
| --- | ---: |
| 把用户假设或模拟数据写成事实 | -10 |
| TAM/SAM/SOM 混淆 | -10 |
| 只按收入或线索判断渠道 | -8 |
| 异常没有三层排查 | -8 |
| 机会没有验证动作或放弃条件 | -6 |
| 月报只有信息汇总，没有经营动作 | -6 |
| 越权替经营层拍板 | -6 |
| 关键数字无来源或无假设 | 每处 -3 |

## 通过线

| 分数 | 结论 |
| ---: | --- |
| 90-100 | 专家雏形，可进入真实业务数据训练 |
| 80-89 | 可上岗雏形，需补 1 轮薄弱项训练 |
| 70-79 | 继续训练，不能上岗 |
| <70 | 回到角色校准和六大技能靶场 |
