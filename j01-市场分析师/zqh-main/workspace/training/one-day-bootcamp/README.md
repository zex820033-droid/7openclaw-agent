# 市场分析师专家虾 1 天速成训练场

## 训练定位

本训练场用于把 `m01 市场分析师` 从“能读任务”压测到“能交付、能解释、能复盘、能复制”的可上岗雏形。

默认靶场：

- 行业：B2B / AI 工具
- 数据：模拟数据 + 标准模板
- 目标：当天完成六大核心任务、8 个对抗案例、1 个结业考核

## 当天必须守住的底线

1. 不虚胖 TAM：所有市场空间必须有边界、公式、假设和来源等级。
2. 不制造增长幻觉：收入、线索、热度必须同时检查利润、留存、转化和可持续性。
3. 不漏报关键异常：异常必须先排数据，再排内部，最后排外部。

## 训练流程

1. 先读 `agents/m01/README.md`、`SOUL.md`、`AGENTS.md`。
2. 读 `day-schedule.md`，确认当天节奏。
3. 用 `task-cards/six-core-tasks.md` 完成六大核心任务。
4. 用 `cases/adversarial-cases.md` 进行误判压测。
5. 用 `expert-expression-template.md` 统一输出表达。
6. 完成 `final-exam.md`。
7. 用 `scorecard.md` 评分，低于 80 分必须返工。
8. 将错误、亮点和可复用方法写入 `review-ledger.md`。

## 当天通过标准

- 六大任务全部有交付物。
- 关键数字溯源率 100%。
- 至少完成 8 个对抗案例。
- 月报必须包含机会清单和行动建议。
- 异常检测必须包含至少 3 个可能原因和验证路径。
- 结业考核总分 >= 80。
