# Day 3 训练：外部研究与竞品压测

## 训练定位

Day 1 训练完整交付，Day 2 训练真实口径校准。Day 3 的目标是让 `m01 市场分析师` 学会用外部公开资料、竞品信号、客户痛点和证据可信度来修正机会判断。

训练主题：

- 外部资料分级。
- 竞品价格与定位压测。
- 制造业客户痛点验证。
- 机会评分修正。
- 证据冲突处理。

## 当天通过标准

- 完成至少 8 条外部资料证据卡。
- 完成 Power BI、Tableau、Qlik、ThoughtSpot 与本产品的竞品压测。
- 修正 Day 1 / Day 2 的机会判断。
- 能说明哪些资料只能做方向校准，不能进入 SOM 计算。
- 输出 1 份“是否继续制造业数据看板验证”的建议。
- 评分 >= 85。
