# Day 3 任务卡

## Task D3-001：外部证据分级

| 字段 | 内容 |
| --- | --- |
| 业务目标 | 判断哪些公开资料能进入市场判断 |
| 输入 | `datasets/external-evidence-matrix.csv` |
| 输出 | 证据分级表 |
| 通过标准 | 每条资料说明可用与不可用边界 |

## Task D3-002：竞品压测

| 字段 | 内容 |
| --- | --- |
| 业务目标 | 识别制造业数据看板的真实竞争压力 |
| 输入 | `datasets/competitor-matrix.csv` |
| 输出 | 竞品威胁等级和反定位 |
| 通过标准 | 包含直接竞品、替代方案、价格压力和实施成本 |

## Task D3-003：机会评分修正

| 字段 | 内容 |
| --- | --- |
| 业务目标 | 用外部证据修正 Day 1 机会排序 |
| 输入 | Day 1 机会评分、Day 2 SOM、Day 3 证据矩阵 |
| 输出 | 修正后的机会评分 |
| 通过标准 | 分数变化必须有证据解释 |

## Task D3-004：证据冲突处理

| 字段 | 内容 |
| --- | --- |
| 业务目标 | 避免不同市场报告口径混用 |
| 输入 | 外部资料矩阵 |
| 输出 | 可用结论 / 不可用结论 |
| 通过标准 | 不平均不同口径，不把全球市场换成目标客户市场 |

## Task D3-005：继续验证建议

| 字段 | 内容 |
| --- | --- |
| 业务目标 | 判断是否继续制造业数据看板验证 |
| 输入 | 外部证据、竞品压测、Day 2 SOM |
| 输出 | 继续 / 收缩 / 转向建议 |
| 通过标准 | 给出验证动作、放弃条件和下次复盘时间 |
