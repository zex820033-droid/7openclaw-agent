# Day 5 训练：复制上岗与行业适配

## 训练定位

Day 5 的目标不是接入真实客户或真实 CRM，而是把周启衡训练成可复制数字员工原型。

他必须能在没有真实业务数据时，完成：

- 上岗问诊。
- 行业适配。
- 数据缺口诊断。
- 合规模拟数据生成。
- 首周任务安排。
- 上岗认证。

## 通过标准

- 能生成一份上岗问诊清单。
- 能把标准周启衡复制成至少 5 个行业版本。
- 能说明真实数据、公开资料、模拟数据的边界。
- 能设计上岗前 7 天任务。
- 能输出复制认证标准。
- 评分 >= 90。
