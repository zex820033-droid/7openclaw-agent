# 周启衡复制上岗问诊清单

## 使用规则

P0 必须在上岗前补齐。P1 可在首周内补齐。P2 可在第二周后逐步完善。

## 业务边界

| 优先级 | 问题 | 用途 | 缺失替代 |
| --- | --- | --- | --- |
| P0 | 目标行业是什么？ | 定义市场边界 | 先选行业模板 |
| P0 | 目标地区是什么？ | 定义市场规模口径 | 使用默认地区假设 |
| P0 | 核心产品/服务是什么？ | 定义产品边界 | 用一句话产品描述 |
| P0 | 目标客户是谁？ | 定义 SAM 和 ICP | 使用行业通用 ICP |
| P0 | 默认币种是什么？ | 统一测算口径 | CNY 或 USD |
| P1 | 价格带/ARPA 是多少？ | 校准收入空间 | 使用行业模拟 ARPA |
| P1 | 当前主要渠道是什么？ | 渠道分析与 SOM | 使用行业渠道模板 |

## 数据源

| 优先级 | 问题 | 用途 | 缺失替代 |
| --- | --- | --- | --- |
| P0 | 是否有 CRM / 客户名单？ | SOM 和漏斗 | 生成模拟目标账户池 |
| P0 | 是否有订单 / 合同数据？ | ARPA、收入、渠道 | 使用模拟订单表 |
| P1 | 是否有用户行为数据？ | 用户分层 | 使用模拟 RFM/行为表 |
| P1 | 是否有渠道成本数据？ | 渠道利润真实性 | 使用模拟 CAC、折扣、服务工时 |
| P1 | 是否有产品埋点？ | 行为变化和异常检测 | 使用模拟活跃和转化数据 |
| P2 | 是否有竞品名单？ | 竞品压测 | 用公开资料和行业默认竞品 |

## 交付与治理

| 优先级 | 问题 | 用途 | 缺失替代 |
| --- | --- | --- | --- |
| P0 | 月报接收人是谁？ | 明确交付对象 | 养虾人 |
| P0 | 告警接收人是谁？ | 明确异常升级 | 养虾人 |
| P0 | 谁验收交付物？ | 三证验真 | 养虾人 |
| P1 | 月报频率是什么？ | 心跳节律 | 月度 |
| P1 | TAM/SAM/SOM 刷新频率是什么？ | 市场空间治理 | 季度 |
| P1 | P1/P2/P3 告警标准是什么？ | 异常边界 | 使用默认阈值 |
