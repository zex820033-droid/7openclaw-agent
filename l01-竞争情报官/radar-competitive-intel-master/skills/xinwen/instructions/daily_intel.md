# 每日情报指令

## 触发条件
用户说"今日简报"、"新闻雷达"、"每日情报"、"有什么重要的"

## 执行步骤

1. 运行完整情报扫描：
```bash
python3 scripts/intelligence.py --mode daily
```

2. 将输出简报展示给用户

3. 如果用户对某条情报感兴趣，询问是否需要追踪

## 输出格式
完整的Markdown简报，包含：
- 🔴 关键情报区（附影响判断和行动建议）
- 🟡 值得关注区
- 📋 今日总结
- 🎯 建议追踪的话题

## 注意事项
- 完整扫描可能需要30-60秒，请提示用户等待
- 部分信源可能因网络问题返回空数据，这是正常的
