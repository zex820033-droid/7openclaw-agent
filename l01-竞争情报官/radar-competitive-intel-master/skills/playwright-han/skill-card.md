## Description: <br>
Playwright Browser Automation helps agents provide Python Playwright guidance and examples for browser automation, screenshots, element waits, network interception, form interaction, and Selenium migration. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers use this skill to learn, create, or migrate Python browser automation workflows with Playwright. It supports common tasks such as browser launch, page navigation, screenshots, search result extraction, form actions, network interception, waits, and Selenium-to-Playwright migration guidance. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Browser automation can interact with real websites, accounts, forms, and captured page content. <br>
Mitigation: Use the skill only on sites and accounts the operator is authorized to automate, and require confirmation before submitting forms or capturing sensitive pages. <br>
Risk: Anti-bot workaround examples can be misused to bypass site rules or access controls. <br>
Mitigation: Do not use anti-bot techniques to evade terms, rate limits, access controls, or site security protections. <br>
Risk: Installing browser automation dependencies can affect the local runtime environment. <br>
Mitigation: Install Playwright and browser dependencies in a virtual environment when possible. <br>
