# A股情报调查员 / ai-intelligence-investigator

---

## 简介

基于 17 个搜索引擎的深度A股情报调查工具，自动编排搜索策略，多源交叉验证消除偏差，生成结构化调查报告。覆盖A股情报调查、竞品分析、舆情监测、人物背景调查、信息真实性验证五大场景，遵循「多源必证、引擎适配、偏差消除」三大原则，帮助用户高效获取可信情报，辅助投资决策。

**核心价值**

- **A股情报调查**：覆盖上市公司基本面、财务数据、行业赛道、概念热度、资金流向、公告研报、股吧舆情等全方位情报。
- **多源交叉验证**：关键信息至少 2 个独立来源确认，避免单一信源偏差。
- **智能引擎编排**：根据调查目标（A股情报、中文舆情、国际视野、隐私敏感、学术验证等）自动选择最优搜索引擎组合。
- **可信度量化**：ABCD 四级信源分级 + ✅/⚠️/❌/🔍 多源确认标注，让信息可信度一目了然。
- **结构化报告**：A股情报、竞品分析、舆情事件、人物背景、信息验证五类报告模板，输出专业、可复用的调查结果。

**适用对象**

- 📈 **A股投资者 / 散户** — 上市公司基本面调查、行业赛道分析、概念热度追踪、机构观点汇总。
- 🏢 **品牌方 / 企业管理者** — 竞品情报调查、市场动态追踪、合作方背景调查。
- 📊 **分析师 / 研究人员** — 多源数据收集、信息交叉验证、结构化报告生成。
- 📰 **媒体 / 内容创作者** — 热点事件溯源、舆论走向分析、信息来源核实。
- 🔒 **法务 / 风控人员** — 人物背景调查、诉讼记录排查、合作风险评估。

---

## 功能特性

### 核心功能

- **A股情报调查**：多引擎搜索上市公司财报、公告、研报、行业政策、资金流向、股吧舆情，辅助投资决策参考。
- **竞品情报调查**：多引擎搜索竞品产品功能、用户口碑、市场表现，输出 SWOT 分析及行动建议。
- **舆情事件调查**：事件还原 + 多视角收集 + 时间线重建，呈现完整舆论图谱。
- **人物背景调查**：身份信息核实、职业履历追溯、专业成就验证、风险信号排查。
- **信息交叉验证**：信息溯源 → 多源比对 → 权威验证，给出已确认 / 待确认 / 已否定的综合判定。
- **引擎自动编排**：根据调查目标智能匹配合适的搜索引擎组合（最多 17 个引擎），消除单一引擎偏差。
- **可信度分级标注**：ABCD 四级信源分级，✅已确认 / ⚠️待确认 / ❌已否定 / 🔍单一来源 四种标注状态。

### 特色亮点

- **17+ 引擎广度覆盖**：Baidu、Google、东方财富、同花顺、雪球、巨潮资讯、新浪财经、证券时报、DuckDuckGo、WeChat、Toutiao、Brave、Bing INT 等，支持按地区、按用途灵活组合。
- **A股专项引擎**：东方财富（行情+财务+股吧）、同花顺（概念+题材+问财AI）、雪球（投资者社区+大V观点）、巨潮资讯（官方公告原文）、新浪财经（事件驱动+情绪监测）、证券时报（政策解读+调查报道）。
- **三原则驱动**：多源必证（≥2 独立来源）、引擎适配（按目标智能编排）、偏差消除（不同引擎/地区交叉比对）。
- **四级信源可信度**：A 级（官方/交易所/权威媒体）、B 级（财经专业平台/行业媒体）、C 级（投资者社区/社交媒体）、D 级（匿名来源），让信息可信度可量化。
- **五类专业报告模板**：A股情报、竞品情报、舆情事件、人物背景、信息验证各配备标准化输出模板，专业可复用。

## 重要数据说明

- 调查结果基于多搜索引擎交叉验证，关键信息至少 2 个独立来源确认后才标注为「已确认」。
- 信源分为 ABCD 四级：A 级（官方/交易所/权威媒体）、B 级（财经专业平台/行业媒体）、C 级（投资者社区/社交媒体）、D 级（匿名/未验证来源）。
- A股调查中，巨潮资讯（cninfo.com.cn）、上交所（sse.com.cn）、深交所（szse.cn）为 A 级信源；东方财富、同花顺、证券时报为 B 级信源；雪球、股吧为 C 级信源。
- 搜索结果受搜索引擎索引时效影响，实时性要求高的调查建议使用时间过滤（如 `tbs=qdr:d`）。
- 调查模式自动适配：A股情报优先 Baidu + 东方财富 + 雪球 + 巨潮资讯；中文舆情优先 Baidu + WeChat + Toutiao；国际视野优先 Google + Brave + Yahoo。
- 所有调查结果均来源于公开可访问的搜索引擎，不涉及非公开数据采集。
- **免责声明**：A股调查报告基于公开信息整理，仅供参考，不构成任何投资建议。股市有风险，投资需谨慎。
