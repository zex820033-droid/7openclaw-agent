## 这个 Agent 能做什么？

Fengniao 是 OpenClaw Agent 编队的**首席竞争情报官**。它的工作不是"搜新闻"，而是**经过验证、附带概率判断、指向行动的洞察**。

| 能力 | 说明 | 你会收到什么 |

| 📡 **竞品自动监控** | 每日扫描 28 家竞品的官网/博客/GitHub/定价页 | 竞品有新功能/融资/定价变化 → 自动推送 |

## 情报质量承诺

每条情报都经过 **六重门禁** 验证：

| 门禁 | 含义 | 标准 |
|:--:|------|------|
| G1 完整性 | 每份情报含"事实+判断+行动"三件套 | 缺一不可 |
| G2 可追溯 | 每条判断有来源指向，标注验证深度 | 来源 A/B/C/D 四级 |
| G3 概率标注 | 不用"可能"，用"置信度 78%" | 每条判断有数字 |
| G4 时效性 | 标注"有效截至时间" | 不过期情报不推送 |
| G5 无冗余 | 战略中枢不知道的信息才推送 | 不重复已知信息 |
| G6 零推测 | 每个数据点有 API/web_fetch 来源 | 无来源不填写 |
