## 四、3条最重要整改动作

1. **固化报销初审四标签边界**
   - 通过：材料齐套 + 金额与审批一致。
   - 退回补证：缺审批 / 缺票 / 缺支付凭证 / 说明不足。
   - 挂起：超标 / 疑似重复 / 预算不足 / 跨期异常，需要多角色判断。
   - 驳回：个人消费 / 虚假材料 / 无法解释。

2. **建立“自评前回读正文”纪律**
   - 写自评或批改前，必须回读自己的正文，避免“正文已写、总结却说遗漏”的自相矛盾。

3. **澄清离职 T 日 / T+1 权限节点**
   - T日：最后工作日，下班前完成资产回收确认。
   - T+1：IT关闭账号的执行节点。
   - 不得表达为员工离职后仍可保留权限。
