## 训练目标

- 熟练运用报销四标签（**通过 / 退回补证 / 挂起 / 驳回**），不混淆。
- 命中 7 项常见风险信号（§4.2）和 7 类不予报销（§3.2）。
- 区分"材料缺失（退回补证）"与"风险叠加需多角色判断（挂起）"。
- 在风险标签和下一步动作上做到可执行、可追踪。

## 作答格式

每题按以下结构作答（强制）：

```markdown
### Qx：场景标题
- 场景要素：申请人 / 角色 / 费用类型 / 关键数字 / 已知材料
- 结论：**通过 / 退回补证 / 挂起 / 驳回**（四选一，必须加粗）
- 依据文件路径：`knowledge/kunjie-admin/xx` §X.X
- 关键规则或数据：（引用原文章节阈值/标准）
- 风险标签：（无 / 超标 / 缺票 / 缺审批 / 缺支付凭证 / 缺客户名单 / 疑似重复 / 逾期 / 抬头错误 / 缺验收 / 个人消费 / 无法说明）
- 下一步动作：（具体到「通知谁、做什么、交付什么」）
- 是否转财务：（是 / 否 / 视情况）
- 三证留痕：审批/单据/回执（题面涉及的必填）
```

## Q1：差旅缺审批单

销售部王然（普通员工）申请报销：
- 北京出差 3 天，机票 ¥1280、火车票 ¥260、出租车 ¥180、酒店 ¥1500（500 元/晚×3 晚）。
- 材料：电子客票行程单、火车票发票、出租车电子发票、酒店发票、支付凭证。
- **没有差旅审批单**。

请判断行政初审结论。

## Q2：酒店超标

销售部李然（销售总监）申请报销：
- 上海出差 2 晚，酒店 ¥1800（**900 元/晚 × 2 晚**），出差前有差旅审批单，备注"陪同重要客户考察"。
- 材料：差旅审批单（标准 650 元/晚，注明"陪同高管客户，预算上浮 30%"）、酒店发票（900 元/晚）、水单、支付凭证。
- 部门负责人审批意见："同意"。

请判断行政初审结论。

## Q3：客户接待缺客户名单

销售部申请报销：

- 发票抬头"鲲界科技有限公司"，有支付凭证。

请判断行政初审结论。

## Q4：发票抬头错误

- 有采购审批单（金额 ¥680，500 元以下，行政自审）、支付凭证、京东订单截图。

请判断行政初审结论。

## Q5：疑似重复报销

销售部赵晴申请报销：

请判断行政初审结论。

## Q6：报销逾期

## Q7：采购报销缺验收单

研发部申请报销一项办公采购：
- 4K 显示器 2 台，单价 ¥3200，合计 ¥6400。
- 走"≥¥5000 档"采购：有三家比价表（SUP-IT-001/002/003）、财务复核、养虾人终审、采购订单。
- **物资已到货并投入使用**，使用人签收。
- 报销时附：发票、支付凭证、采购审批单、比价表。
- **缺少独立的"验收单/入库单"**（无单独签字验收记录）。

请判断行政初审结论。

## 评分重点

| 维度 | 权重 | 评分标准 |
|------|:----:|----------|
| 标签定性准确 | 40% | 通过 / 退回补证 / 挂起 / 驳回 不能混用，关键题错一个标签扣 10-15 分 |
| 依据路径可追溯 | 20% | 必须给出文件路径 + 章节号 |
| 风险标签不漏不滥 | 15% | 命中核心风险信号（§4.2 七项 + §3.2 七类） |
| 下一步可执行 | 15% | 明确通知谁、做什么、交付什么 |
| 三证/转财务判断 | 10% | 与标签定性一致；不允许「挂起+转财务」或「退回+转财务」 |

## 及格线

- **75 分**：主标签全部正确
- **90 分**：标签 + 风险信号 + 三证全部到位
- **100 分**：能指出题目之间风险叠加的边界（例如 Q2 单项超标有审批 vs Q3 缺客户名单的差异）
