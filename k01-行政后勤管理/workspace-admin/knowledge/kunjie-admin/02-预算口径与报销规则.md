> 版本：v1.0
> 适用期间：2026 年度训练口径
> 说明：本文件用于训练行政虾做报销初审、预算预警和财务接口，不替代财务最终核算。

## 1.2 月度预算复盘字段

行政每月 5 日前输出上月行政预算复盘，至少包含：

- 各科目预算、实际、执行率。
- 超 80% 科目的原因。
- 单笔超过 5000 元支出明细。
- 供应商价格波动。
- 下月已知大额支出。
- 可节省机会和需要养虾人决策的事项。

## 2. 预算预警阈值

| 状态 | 触发条件 | 行政动作 |
|------|----------|----------|
| 绿色 | 月度执行率 < 70% | 正常记录 |
| 黄色 | 月度执行率 70%-90% | 提醒部门负责人和财务 |
| 橙色 | 月度执行率 90%-100% | 后续支出需额外说明 |
| 红色 | 超预算 | 挂起新增支出，提交养虾人确认 |

## 3. 报销材料标准

| 费用类型 | 必备材料 | 常见退回原因 |
|----------|----------|--------------|
| 差旅机票 | 差旅审批单、电子客票行程单、支付凭证 | 无事前审批、舱位超标 |
| 火车票 | 差旅审批单、电子发票或行程凭证 | 日期与行程不符 |
| 酒店 | 差旅审批单、酒店发票、水单或订单截图 | 城市标准超额无说明 |
| 市内交通 | 交通订单、发票、行程说明 | 非工作时间无说明 |
| 客户餐饮 | 接待审批、发票、客户名单、商务目的 | 人均超标、无客户信息 |
| 办公采购 | 采购审批、报价单、发票、验收单、入库单 | 缺验收、未比价 |
| 快递邮寄 | 快递面单、发票、业务归属 | 个人件混入公司报销 |
| 维修维保 | 报修工单、报价、维修记录、验收照片 | 未留维修前后证据 |

## 3.1 发票要求

| 项目 | 要求 |
|------|------|
| 发票抬头 | 鲲界科技有限公司 |
| 税号 | 训练数据不写真实税号，真实落地时由财务提供 |
| 发票类型 | 电子普票/专票按财务要求执行 |
| 金额 | 必须与报销金额一致；差额需说明 |
| 日期 | 原则上应在业务发生当月或次月 |
| 附件 | 发票、支付凭证、审批单、业务说明 |

## 3.2 不予报销清单

- 个人娱乐、个人购物、非工作餐饮。
- 无法说明业务目的的礼品。
- 非公司员工个人物品，未获审批的除外。
- 违反差旅标准且无事前审批的超额部分。
- 发票抬头错误且无法重开的费用。
- 已由客户、供应商或第三方承担的费用。
- 明显重复提交的费用。

## 4. 报销时限

- 普通报销：费用发生后 30 天内提交。
- 差旅报销：返程后 7 个自然日内提交。
- 跨年费用：原则上不得跨年报销；特殊事项需财务确认。
- 逾期提交：需写明原因，由部门负责人确认后进入财务复核。

## 5. 费用标准

### 5.1 差旅住宿

| 城市等级 | 员工标准 | 经理/总监标准 | 高管/客户陪同 |
|----------|----------|---------------|---------------|
| 一线城市 | 500 元/晚 | 650 元/晚 | 900 元/晚以内审批 |
| 新一线城市 | 420 元/晚 | 500 元/晚 | 750 元/晚以内审批 |
| 其他城市 | 320 元/晚 | 380 元/晚 | 600 元/晚以内审批 |

### 5.2 餐饮接待

| 场景 | 标准 | 审批 |
|------|------|------|
| 内部加班餐 | 50 元/人以内 | 部门负责人 |
| 普通客户接待 | 150 元/人以内 | 部门负责人 + 行政备案 |
| 重要客户接待 | 300 元/人以内 | 养虾人或授权负责人 |
| 超标准接待 | 逐笔说明 | 养虾人终审 |

### 5.3 市内交通

- 工作日 22:00 后加班返程可报销网约车。
- 跨办公点会议可报销公共交通或企业网约车。
- 非工作目的绕行费用不予报销。
- 同路线多人同行应优先拼车，行政可提示异常。

### 5.4 通讯与居家办公

| 类型 | 标准 | 说明 |
|------|------|------|
| 公司手机卡 | 特定岗位审批配置 | 销售、客服值班、高管助理 |
| 居家办公设备 | 按资产领用，不走个人报销 | 显示器、键鼠、摄像头需登记 |
| 网络补贴 | 默认无固定补贴 | 特殊岗位需养虾人审批 |

### 5.5 礼品与纪念品

| 场景 | 标准 | 材料 |
|------|------|------|
| 客户伴手礼 | 300 元/人以内 | 客户名单、商务目的、采购审批 |
| 活动纪念品 | 100 元/人以内 | 活动方案、名单、验收照片 |
| 员工关怀 | 200 元/人以内 | HR 方案、名单 |

礼品不得采购现金卡、购物卡等高合规风险物品，特殊情况必须由养虾人终审。

## 6. 超标处理

| 情况 | 处理 |
|------|------|
| 有事前审批且理由充分 | 行政标记“已审批超标”，转财务 |
| 无事前审批但业务必要 | 退回补说明，部门负责人确认后转财务判断 |
| 无审批且理由不充分 | 行政初审不通过 |
| 疑似重复/拆单/个人消费 | 挂起，上报养虾人和财务 |

|------|------|

| 行政备注 | 上海-北京客户拜访，材料齐套 |

## 7.1 行政与财务月结对账

每月 3 日前，行政向财务提供：

- 上月采购验收清单。
- 上月资产入库与调拨清单。
- 上月报销初审退回/挂起明细。
- 合同到期与供应商续约提示。
- 预算执行异常说明。

财务向行政反馈：

- 付款完成状态。
- 发票异常清单。
- 科目归属调整。
- 需要行政补充的验收或归档材料。

## 8. 行政不得做的事

- 不承诺付款日期。
- 不替申请人编造业务目的。
- 不修改发票信息。
- 不把无审批事项包装为已审批。
- 不绕过财务处理付款、借款或冲账。
