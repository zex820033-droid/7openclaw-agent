## 3. 账号申请流程

1. HRBP 在入职前 2 个工作日发《入职后勤需求单》。
2. 行政确认工位、门禁、办公用品和系统清单。
3. IT 开通企业邮箱和 SSO。
4. 行政按岗位开通差旅/报销/会议权限。
5. 员工入职当天完成账号登录确认。
6. 行政在入职后勤台账记录状态。

### 3.1 入职后勤清单

| 时间 | 行政动作 |
|------|----------|
| 入职前 T-2 | 收到 HRBP 名单，确认岗位、部门、办公点 |
| 入职前 T-1 | 准备工位、电脑、显示器、门禁、办公用品 |
| 入职当天 | 引导门禁、访客转员工、确认系统能登录 |
| 入职后 T+1 | 确认资产签收、差旅/报销权限是否需要开通 |
| 入职后 T+7 | 回访办公体验，处理设备或工位问题 |

## 4. 离职回收流程

| 时间点 | 行政动作 |
|--------|----------|
| T-3 工作日 | 接收 HRBP 离职通知，列出资产和系统权限 |
| T-1 工作日 | 确认待归还设备、门禁、办公用品 |
| T 日 | 回收资产，更新责任人和状态 |
| T+1 工作日 | 通知 IT 关闭账号，取消商旅/用车权限 |
| T+3 工作日 | 完成资产盘点和异常上报 |

### 4.1 离职资产回收清单

- 笔记本电脑、显示器、键鼠、转接头。
- 门禁卡、工牌、停车权限。
- 公司手机、电话卡、测试设备。
- 纸质合同、印章借用文件、保密资料。
- 差旅、用车、报销、会议、访客系统权限。

## 6. 安全与审计

- 所有管理员权限必须启用 MFA。
- 账号权限按岗位授予，离岗即回收。
- 行政每月导出一次系统权限清单，与 HRBP 名单核对。
- 发现离职人员仍有权限，按 P1 风险上报。
- 发现共享账号、明文密码、非授权登录，立即停用并上报。

## 8. 禁止事项

- 禁止在文档、聊天、邮件正文中明文发送密码。
- 禁止共用管理员账号。
- 禁止为未入职人员提前开通报销、差旅、用车权限。
- 禁止保留离职员工商旅和用车权限。
- 禁止行政代替审批人批准自己的申请。
