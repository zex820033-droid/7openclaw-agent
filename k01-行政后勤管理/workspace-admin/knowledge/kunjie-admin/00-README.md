## 行政虾的知识使用原则

1. 先查 `00-README.md` 确认公司、目录和数据性质。
2. 日常流程查 `01-行政SOP.md`。
3. 费用、预算、报销、超标查 `02-预算口径与报销规则.md`。
4. 资产问题查 `03-资产台账.csv`，以资产编号为唯一主键。
5. 账号、权限、离职回收查 `04-差旅平台-OA-报销系统账号.md`。
6. 采购选型、合同到期、供应商风险查 `05-供应商清单.csv`。
7. 出题、考试、复盘查 `06-训练任务建议.md` 和 `exam/`。

## k01 启动时应先记住的 12 条事实

1. 鲲界科技行政当前服务 82 人，主办公点在张江，另有静安共享会议点。
2. 行政工作必须留下三证：申请/审批记录、交付/验收记录、台账/归档记录。
3. 采购金额大于等于 5000 元必须三家比价或招采说明，并由财务复核后提交养虾人终审。
4. 差旅实行“先审批、后预订、再报销”，无事前审批的超标费用默认挂起。
5. 报销材料以电子发票、行程单、支付凭证、审批单为基础，缺一项需补证。
6. 资产以资产编号为唯一主键，做到卡、物、账、责任人四相符。
7. 资产调拨、报修、报废都必须更新台账，不能只在聊天里确认。
8. 办公用品设置安全库存，低于安全库存触发补货建议。
9. 供应商按 A/B/C 分级，A级优先，C级只能应急或需额外审批。
10. 行政虾不做付款和财务核算，只做单据初审、凭证齐套、台账更新和风险提示。
11. 系统账号只记录入口、角色和权限，不记录密码；密码托管在企业密码库。
12. 所有训练数据在真实使用前必须由养虾人确认，不能把合成数据当真实制度。
