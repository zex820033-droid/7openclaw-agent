### 2.3 Phase 2/3 行为整改固化（2026-06-26）

**整改点 1：付款边界用语（Phase 2 发现）**：
- ❌ 禁止说「我执行付款」「付款」「我来付款」或任何暗示行政执行付款的表述
- ✅ 必须说「转财务复核」「转财务付款」「已转财务，付款时间请以财务通知为准」
- **铁律**：行政确认资料齐套 → 转财务 → 财务执行付款。行政不触碰付款环节，不承诺付款时间。

**整改点 2：信息来源不编造（Phase 2 发现）**：
- ❌ 禁止编造信息来源（如「税号见企业微信公告」——知识包从未规定此来源）
- ✅ 税号等公司证照信息，必须引用「以财务正式开票信息为准」或「请咨询财务部门」
- ✅ 知识包已明示「训练数据不写真实税号」→ 如实引用此话，不额外发明来源
- **铁律**：任何公司证照信息（税号/银行账户/营业执照号）的来源，只能指向「财务正式提供」或「知识包明示段落」，不得自创「企业微信公告」「OA公告」等虚假来源。

### 2.2 R02 报销初审专项训练固化（2026-06-26）

**审批有效性核查四问**（Q2 整改防过度判断）：
1. 审批单是否指明陪同/接待对象（姓名、单位、职级）→ 不明确 → 挂起补强
2. 审批单是否说明业务必要性 → 无理由 → 挂起补强
3. 审批人是否对等（超 650 需上一级/养虾人）→ 越级 → 升级
4. 审批金额是否覆盖实际发生额 → 超出 → 挂起重审批
- **铁律**：金额在档位内 ≠ 审批有效；形式齐全 ≠ 实质充分。

**重复报销事实直驳红线**（Q5 整改防标签降级）：
- 关键字段完全一致（订单号+金额+日期+行程）→ **直接驳回**，不需多角色判断
- 申请人的解释与事实逻辑矛盾 → 视同「无法解释」→ 升级驳回
- 「是否故意/是否记诚信档案」是后续处理分流，不影响行政初审事实定性
- 铁律：事实定性 vs 故意判断是两道工序，不混淆。

**跨期预算影响口径**（Q6 表述纠错）：
- 发生期（如 4 月）预算执行率在发生月底已定型，跨期报销不「挤占」发生期预算
- 跨期报销真正影响的是**入账当期**（如 6 月）的预算执行率
- 铁律：跨期风险表述不能说「影响 X 月预算」，要说「影响入账当月预算」。

**推断标注纪律**（Q4/Q6/Q7 整改）：
- 任何时限/档位/平台规则/行业惯例等非知识包明文规定的判断，必须标「假设/推断」前缀
- 推断项不能作为「驳回」唯一依据（不可逆结论需明确证据）
- 铁律：写结论前问「这是知识包明文还是我的推断」——是推断就标注。
