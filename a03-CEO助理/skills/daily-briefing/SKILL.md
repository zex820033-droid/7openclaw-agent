# 每日简报 (Daily Briefing)

早晨生成当日工作简报，让养虾人 30 秒看清今天要做什么。

## 触发条件

- "早上好" / "早安" / "morning"
- "今天有什么"
- "给我一个 daily briefing"
- 工作日开始时自动触发

## 简报内容

### 1. 日历概览
今天有什么会？几点到几点？
（如果能接入日历，直接拉取；否则提醒养虾人手动确认）

### 2. 今日待办
- 截止日期是今天的所有任务
- 进行中的高优先级任务
- 昨天未完的延续事项

### 3. 阻塞项
- 有哪些任务被卡住了、等什么
- 需要养虾人拍板的事项（如果有）

### 4. 需要关注
- 虾群汇报的重要信息
- 数据异常预警（如果有）
- 竞品动态（如果有重要变化）

## 输出格式

```markdown
☀️ 早上好，[名字]。今天是 YYYY年MM月DD日 [周X]。

## 📅 今日日程
| 时间 | 事项 |
|------|------|
| 10:00 | XX会议 |
| 14:00 | XX评审 |

## 📋 待办优先级
🔴 **必须今天完成**
1. [任务] — 截止今天

🟡 **尽量推进**
2. [任务] — 进度 60%

## 🚧 阻塞中
- [任务]：等 XX 回复（已等 3 天）

## ⚡ 速报
- a02 提示：本月 GMV 环比 -8%，建议今天看下
- b01 发现：竞品 XX 上线了新功能

今天从 [最高优先级任务] 开始？
```

## 注意事项
- 压缩到 30 秒能读完
- 只列真正的待办，不要列 15 项 —— 最多 5 项
- 阻塞项要说明等什么、等了多久
- 没有的事不要编（没日历数据就老实说没接入）
