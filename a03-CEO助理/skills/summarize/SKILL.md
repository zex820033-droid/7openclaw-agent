# 摘要生成 (Summarize)

把任何长内容压缩成精炼摘要。养虾人没时间读长篇大论。

## 触发条件

- "帮我总结一下"
- "这个太长，给个摘要"
- "要点是什么"

## 摘要层级

按养虾人需要选择深度：

### 一句话摘要（10 秒）
适合：快速扫描
```
这篇文章说 X 公司的 Y 策略因为 Z 原因失败了。
```

### 要点摘要（30 秒）
适合：需要知道关键信息
```
1. 核心发现：[一句话]
2. 为什么重要：[一句话]
3. 跟我们的关系：[一句话]
```

### 详细摘要（2 分钟）
适合：需要做决策的依据
```
## 背景
[为什么要关注这个]

## 关键观点
1. [观点 + 支撑论据]
2. [观点 + 支撑论据]

## 对我们的影响
[和我们业务的关联]

## 建议
[要不要进一步了解/采取行动]
```

## 输出格式

无论用哪一级，结尾加一行：
```
📎 来源：[URL] | 原文约 X 字 → 压缩至 Y 字
```

## 注意事项
- 不要引入原文没有的信息
- 不确定的地方标注"（原文未明确）"
- 如果是中文内容，摘要也用中文；英文原文保留关键术语
- 多个来源时，标注每条摘要对应的来源
