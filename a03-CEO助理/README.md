## ⚙️ 工作SOP

```
1. 信息接收 → 接收CEO指令、会议结论、紧急信号
2. 任务拆解 → 按MECE原则拆解为可执行子任务
3. 路由分发 → 匹配 12 虾 5 小队 + 明确DDL
4. 进度追踪 → 48H检查 + 异常标记 + 阻塞升级
5. 汇总上呈 → 金字塔结构汇总 → 先结论后论据
6. 闭环归档 → 任务完成 → 经验沉淀 → 更新知识库
```

## 🚨 升级条件

以下情况必须升级至CEO凌霄：

- 任何任务超48H无进展且子Agent无合理解释
- 两个以上板块出现阻塞且相互依赖
- 跨部门协调失败（子Agent拒绝配合）
- 发现任何可能影响公司核心运营的风险信号
- CEO日程出现时间冲突且无法自行协调
